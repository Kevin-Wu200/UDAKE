# 地图引擎架构说明

## 概述

本次更新实现了统一的地图引擎抽象层，支持天地图和 ArcGIS 两种地图引擎，并添加了多项增强功能。

## 核心架构

### 1. BaseMapEngine（基础引擎接口）
位置：`frontend/js/map/core/BaseMapEngine.js`

定义统一的地图操作接口：
- `init(container, options)` - 初始化地图
- `setCenter(center)` - 设置中心点
- `getCenter()` - 获取中心点
- `setZoom(zoom)` - 设置缩放级别
- `getZoom()` - 获取缩放级别
- `fitToBounds(bounds)` - 适配到指定边界
- `onZoom(callback)` - 注册缩放回调
- `onMove(callback)` - 注册移动回调
- `destroy()` - 销毁实例

能力标记：
- `supportsCustomReset` - 是否支持自定义 reset 按钮

### 2. TiandituEngine（天地图引擎）
位置：`frontend/js/map/core/TiandituEngine.js`

特性：
- `supportsCustomReset = true`
- 左键拖动地图
- 自研 WMTS 瓦片渲染
- 鼠标滚轮缩放
- 瓦片缓存和错误重试

### 3. ArcGISEngine（ArcGIS 引擎）
位置：`frontend/js/map/core/ArcGISEngine.js`

特性：
- `supportsCustomReset = false`
- 使用 ArcGIS 内置 Home 控件
- 自动深色模式切换
- 原生 ArcGIS API 支持

## 管理器和组件

### 1. MapManager（地图管理器）
位置：`frontend/js/managers/MapManager.js`

职责：
- 根据配置创建对应的地图引擎
- 管理地图状态（initialCenter, initialZoom, geojson, mode）
- 条件渲染 reset 按钮（仅天地图）
- 处理模式切换（normal / areaSampling）

Reset 按钮逻辑：
- 普通模式：回到初始位置
- 区域采样模式：适配 GeoJSON 范围

### 2. ZoomControl（缩放控件）
位置：`frontend/js/components/ZoomControl.js`

特性：
- 右下角动态显示
- 监听缩放变化更新滑块位置
- 显示后 1 秒自动淡出
- 半透明背景，平滑动画

### 3. GeoUtils（地理工具）
位置：`frontend/js/utils/GeoUtils.js`

功能：
- `calculateBoundsFromGeoJSON(geojson)` - 计算 GeoJSON 边界
- `extractCoordinates(geometry)` - 提取几何坐标
- `calculateDistance(point1, point2)` - 计算两点距离
- `expandBounds(bounds, padding)` - 扩展边界

## 适配器更新

### TiandituAdapter 和 ArcGISAdapter
- 内部使用新的 Engine 架构
- 保持向后兼容
- 添加 `getEngine()` 方法获取引擎实例

## 使用示例

### 基础使用（保持原有方式）
```javascript
import { initializeMap } from './地图初始化.js';

const adapter = await initializeMap('viewDiv');
const view = adapter.getView();
```

### 使用新引擎架构
```javascript
import { initializeMapWithEngine } from './地图引擎集成.js';

const { manager, zoomControl, engine } = await initializeMapWithEngine('viewDiv', {
    center: [116.39, 39.9],
    zoom: 5
});

// 进入区域采样模式
manager.enterAreaSamplingMode(geojson);

// 退出区域采样模式
manager.enterNormalMode();
```

### 直接使用引擎
```javascript
import { TiandituEngine } from './map/core/TiandituEngine.js';
import { ZoomControl } from './components/ZoomControl.js';

const engine = new TiandituEngine();
await engine.init('viewDiv', { center: [116.39, 39.9], zoom: 5 });

const zoomControl = new ZoomControl(engine);
zoomControl.create('viewDiv');

// 监听缩放变化
engine.onZoom((zoom) => {
    console.log('缩放级别:', zoom);
});
```

## 向后兼容

所有现有代码无需修改即可继续使用。新的引擎架构作为底层实现，通过适配器保持接口一致性。

## 扩展性

要添加新的地图引擎：
1. 继承 `BaseMapEngine`
2. 实现所有抽象方法
3. 设置 `supportsCustomReset` 标记
4. 在 `MapManager` 中添加创建逻辑
