# 部署说明

## Docker部署

### 前置要求

- Docker 20.10+
- Docker Compose 2.0+

### 部署步骤

1. 进入部署目录
```bash
cd deployment
```

2. 启动服务
```bash
docker-compose up -d
```

3. 查看日志
```bash
docker-compose logs -f
```

4. 停止服务
```bash
docker-compose down
```

### 访问地址

- 前端: http://172.20.10.2
- API文档: http://172.20.10.2/api/docs

## 本地开发部署

### 后端部署

1. 安装依赖
```bash
pip install -r requirements.txt
```

2. 启动服务
```bash
cd backend
python -m app.main
```

### 前端部署

直接在浏览器中打开 `frontend/index.html`

或使用简单HTTP服务器:
```bash
cd frontend
python -m http.server 8080
```

## 生产环境配置

1. 修改 `生产环境变量.env`
2. 配置HTTPS证书
3. 调整nginx配置
4. 设置防火墙规则

## 性能优化

- 增加worker进程数
- 启用Redis缓存
- 配置CDN加速
- 数据库连接池

## 监控与日志

- 日志位置: `/var/log/udake/`
- 监控端点: `/health`
- 性能指标: Prometheus + Grafana

## 备份策略

- 数据文件: 每日备份
- 结果文件: 定期清理
- 配置文件: 版本控制
