# 运维培训手册（部署和运维阶段1）

## 培训目标
- 掌握部署脚本全流程。
- 能独立完成监控告警配置与验证。
- 能独立执行日志排查与回滚恢复。

## 培训内容
1. 部署脚本演练：`install -> configure -> start -> update -> stop`
2. 配置校验演练：`deployment/config/validate_config.sh`
3. 告警演练：`deployment/scripts/test_alerts.sh`
4. 日志演练：`deployment/scripts/test_logging.sh`
5. 回滚演练：`deployment/scripts/rollback.sh`

## 考核标准
- 15 分钟内完成一次标准发布。
- 10 分钟内定位一次模拟 5xx 告警。
- 20 分钟内完成一次回滚并恢复健康检查。
