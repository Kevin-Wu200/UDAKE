# 不确定性量化模型说明

## 概述

不确定性量化是空间决策的关键环节，本系统提供完整的不确定性分析框架。

## 不确定性来源

### 1. 测量不确定性
- 仪器误差
- 人为误差
- 环境干扰

### 2. 空间不确定性
- 采样密度不足
- 空间异质性
- 边界效应

### 3. 模型不确定性
- 变异函数拟合误差
- 插值方法假设
- 参数估计误差

## 不确定性指标

### 1. 克里金方差

**定义**: 预测位置的估计方差

**计算**:
```
σ²(s0) = c0 + c - Σλi * γ(si, s0) + μ
```

**解释**:
- 值越大，不确定性越高
- 反映预测可靠性
- 用于置信区间计算

### 2. 置信区间

**95%置信区间**:
```
[Z(s0) - 1.96σ(s0), Z(s0) + 1.96σ(s0)]
```

**应用**:
- 风险评估
- 决策阈值分析
- 采样优化

### 3. 风险指数

**计算公式**:
```
Risk = 0.7 * (σ²/σ²_max) + 0.3 * (|Z|/|Z|_max)
```

**分级**:
- 低风险: Risk < 0.33
- 中风险: 0.33 ≤ Risk < 0.67
- 高风险: Risk ≥ 0.67

## 不确定性分级

### 等级定义

| 等级 | 归一化方差范围 | 颜色 | 建议 |
|-----|--------------|------|------|
| 很低 | [0, 0.2) | 绿色 | 可直接使用 |
| 低 | [0.2, 0.4) | 蓝色 | 可使用 |
| 中等 | [0.4, 0.6) | 橙色 | 谨慎使用 |
| 高 | [0.6, 0.8) | 红色 | 需补充采样 |
| 很高 | [0.8, 1.0] | 深红 | 不建议使用 |

## 不确定性传播

### 误差传播公式

对于函数 f(Z):
```
Var[f(Z)] ≈ (∂f/∂Z)² * Var[Z]
```

### 蒙特卡洛模拟

1. 从预测分布采样N次
2. 计算每次采样的函数值
3. 统计函数值的分布

## 决策阈值分析

### 误判风险

**假阳性风险**:
```
P(Z < threshold | Z_pred > threshold)
```

**假阴性风险**:
```
P(Z > threshold | Z_pred < threshold)
```

### 最优阈值选择

基于ROC曲线和成本函数：
```
Cost = w1 * FP + w2 * FN
```

## 质量控制

### 验证指标

1. **交叉验证RMSE**: < 预测值标准差的30%
2. **方差合理性**: 方差均值 < 预测值方差
3. **空间连续性**: 方差空间分布合理

### 改进建议

- 增加采样密度
- 优化变异函数模型
- 使用更高级的克里金方法
- 结合辅助变量
