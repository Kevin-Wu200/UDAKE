# 更新日志

## 2026-03-03

### 高德地图迁移与标准化

- 将地图引擎迁移至高德地图 JS API v2.0
- 创建独立配置文件 `amap.config.js`，从 HTML 中分离地图配置
- 严格按照官方标准结构实现地图初始化
- 支持用户设备自动定位，定位失败时回退到默认位置
- 修复 Electron 环境中高德地图加载超时问题（调整 webSecurity 配置）
- 增强加载错误诊断功能，添加详细的加载进度日志

### 地图引擎统一抽象层

- 实现 BaseMapEngine 基础引擎接口
- 实现 TiandituEngine 天地图引擎（自研 WMTS 瓦片渲染）
- 实现 ArcGISEngine（集成 Home 控件、深色模式自动切换）
- 实现 MapManager 地图管理器（条件渲染 reset 按钮、模式切换）
- 实现 ZoomControl 动态缩放条组件
- 实现 GeoUtils 地理工具类
- 更新适配器使用新引擎架构，保持向后兼容

## 2026-03-02

### 地图引擎切换功能

- 采用适配器模式实现地图引擎可切换架构
- 支持 ArcGIS 和天地图两种地图引擎
- 创建 MapAdapter 统一接口（13 个方法）
- 实现 ArcGISAdapter 和 TiandituAdapter
- 重构地图初始化、图层管理、主程序模块
- 添加引擎切换脚本 `switch-map-engine.sh`
- 安装 Leaflet 1.9.4 作为天地图底层引擎
- 增强坐标转换功能（WGS-84 与 Web Mercator 互转）

### 网格分辨率输入校验

- 将网格分辨率输入框从 number 类型改为 text 类型，移除上下调节按钮
- 添加实时输入校验，仅允许大于 0 的正整数（1-10000）
- 禁止小数、负数、科学计数法、字母、空格等非法输入
- 在开始插值前进行二次校验

### 区域采样功能

- 实现新建项目功能（自由采样 / 区域采样）
- 支持手动输入坐标和自动定位两种模式
- 实现 GeoJSON 边界上传与点在面内检测
- 添加定位权限管理和错误处理系统
