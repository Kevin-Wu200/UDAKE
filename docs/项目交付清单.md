## 🔧 使用说明

### 1. 切换地图引擎

**方法 1：使用脚本**
```bash
./switch-map-engine.sh tianditu  # 切换到天地图
./switch-map-engine.sh arcgis    # 切换到 ArcGIS
```

**方法 2：手动编辑**
编辑 `frontend/js/config/map.config.js`：
```javascript
MAP_PROVIDER: 'tianditu'  // 或 'arcgis'
```

### 2. 配置 Token（可选）

**天地图：**
编辑 `frontend/js/config/tianditu.config.js`
```javascript
TOKEN: 'YOUR_TIANDITU_TOKEN_PLACEHOLDER'
```

**ArcGIS：**
编辑 `frontend/js/config/arcgis.config.js`
```javascript
API_KEY: "YOUR_ARCGIS_API_KEY_PLACEHOLDER"
```

### 3. 启动应用
```bash
cd dist
npm start
```

### 4. 运行测试
在浏览器中打开：
```
frontend/test-map-engine.html
```
