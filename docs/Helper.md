### Q: UDAKE 是什么？
UDAKE（Uncertainty-Driven Adaptive Kriging Engine）是一个智能不确定性驱动空间决策平台。

**A:** UDAKE 是一个跨平台的空间数据分析平台，支持 macOS、Windows、Web 和 Android。它提供空间插值（克里金）、采样优化、深度学习异常检测、不确定性分析等核心功能，帮助用户做出数据驱动的空间决策。

### Q: 如何开始使用 UDAKE？
我刚下载了 UDAKE，应该从哪里开始？

**A:** 使用 UDAKE 的基本步骤：
1. 启动应用后，点击"新建项目"或打开已有项目
2. 上传您的空间采样数据（支持 CSV、Excel、Shapefile 等格式）
3. 配置克里金插值参数（选择模型类型、设置网格分辨率等）
4. 点击"开始插值"执行计算
5. 在地图上查看插值结果，导出报告

### Q: 支持哪些克里金插值方法？
**A:** UDAKE 支持多种克里金插值方法：
- 普通克里金（Ordinary Kriging）
- 简单克里金（Simple Kriging）
- 泛克里金（Universal Kriging）
- 时空克里金（Spatiotemporal Kriging）
- 3D 克里金（3D Kriging）
- 协同克里金（Co-Kriging）
- 贝叶斯克里金（Bayesian Kriging）

系统会根据数据特征自动推荐最适合的模型。

### Q: 如何看待插值结果的不确定性？
**A:** UDAKE 提供完整的不确定性分析工具：
1. 查看插值结果的方差图（Variance Map）
2. 使用不确定性分级功能，识别高/中/低不确定性区域
3. 查看风险指数报告，了解决策风险
4. 使用异常检测功能，发现潜在的异常采样点
5. 系统支持 SHAP/LIME 可解释性分析，帮助理解模型行为

### Q: 如何加速大规模数据的插值计算？
**A:** 有几个方法可以加速计算：
1. 启用 GPU 加速（需要 NVIDIA GPU + CUDA）：在设置中开启 GPU 加速选项
2. 使用分布式计算：将任务分解到多个计算节点
3. 调整网格分辨率：适当降低分辨率可显著减少计算量
4. 使用批量插值：一次性提交多个区域的计算任务
5. 开启计算缓存：避免重复计算相同参数的任务

### Q: UDAKE 支持哪些数据格式？
**A:** UDAKE 支持以下数据格式：
- CSV（逗号分隔值）
- Excel（.xlsx, .xls）
- Shapefile（.shp）
- GeoJSON（.geojson）
- GeoTIFF（.tif, .tiff）
- KML/KMZ

上传数据时，系统会自动检测坐标列和值列。

### Q: 如何进行采样点优化？
**A:** UDAKE 的采样优化功能包括：
1. 自适应采样建议：根据现有数据的不确定性分布，推荐新的采样位置
2. 采样点影响评估：评估每个采样点对整体插值精度的贡献
3. 强化学习采样：使用 AI 驱动的采样策略
4. 航测像片采样：针对无人机航测数据优化像片选择

### Q: 如何配置 GPU 加速进行克里金插值？
如何配置 GPU 加速进行克里金插值？

**A:** 配置 GPU 加速的步骤：1. 确保安装了 NVIDIA GPU 和 CUDA 驱动；2. 在 UDAKE 设置面板中开启 GPU 加速选项；3. 选择需要加速的克里金模型；4. 提交计算任务时系统会自动使用 GPU。注意：首次使用可能需要下载 CUDA 内核。

### Q: 如何导出 Shapefile 格式的结果？
如何导出 Shapefile 格式的结果？

**A:** 导出 Shapefile 格式的步骤：1. 在结果面板点击导出按钮；2. 选择 Shapefile 格式；3. 指定保存路径；4. 确认导出。导出的文件包含 .shp、.shx、.dbf 三个文件。
