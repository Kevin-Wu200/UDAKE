# 时空克里金模块升级指南

更新时间：2026-04-05

## 1. 升级准备
- 确认变更范围（API/DB/配置）
- 执行备份：`deployment/spatiotemporal_kriging/scripts/backup.sh`
- 确认最近一次备份可校验通过

## 2. 升级步骤
1. 拉取新代码
2. 如有新配置，更新 `.env`
3. 执行：`deployment/spatiotemporal_kriging/scripts/update.sh`
4. 验证健康检查与关键 API

## 3. 回滚方法
- 升级失败后执行：`deployment/spatiotemporal_kriging/scripts/restore.sh <最近备份目录>`
- 再次检查 `/health` 与关键接口

## 4. 验证方法
- `curl -f http://127.0.0.1/health`
- `curl -f https://<域名>/api/startup/health`
- 冒烟接口：`/api/spatiotemporal/predict`
