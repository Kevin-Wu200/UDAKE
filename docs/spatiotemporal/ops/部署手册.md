# 时空克里金模块部署手册

更新时间：2026-04-05

## 1. 环境准备
- 系统：Ubuntu 22.04+
- 资源：CPU 4核，内存 8GB，SSD 100GB（推荐）
- 基础软件：Docker、Docker Compose、Nginx、Python 3.10+、Node.js 18+
- 执行初始化：`deployment/spatiotemporal_kriging/scripts/setup_server.sh`

## 2. 部署步骤
1. 进入目录：`cd deployment/spatiotemporal_kriging`
2. 复制变量模板：`cp .env.example .env`
3. 填写数据库、Redis、JWT、地图 API 密钥
4. 放置证书到 `certs/fullchain.pem` 和 `certs/privkey.pem`
5. 启动部署：`./scripts/deploy.sh`

## 3. 验证方法
- 服务状态：`docker compose -f docker-compose.yml ps`
- 后端健康：`curl -f http://127.0.0.1/health`
- 接口探活：`curl -f https://<域名>/api/startup/health`
- 日志检查：`docker compose -f docker-compose.yml logs --tail=200 backend nginx`

## 4. 常见问题
- 证书路径错误：检查 `certs/` 下文件名是否与 Nginx 配置一致
- 数据库连接失败：确认 `.env` 中 `DATABASE_URL` 与 `AUTH_DATABASE_URL`
- 前端白屏：检查前端构建是否成功，执行 `docker compose -f docker-compose.yml logs frontend`
