# 时空克里金模块运维手册

更新时间：2026-04-05

## 1. 日常运维
- 查看服务：`docker compose -f deployment/spatiotemporal_kriging/docker-compose.yml ps`
- 重启后端：`docker compose -f deployment/spatiotemporal_kriging/docker-compose.yml restart backend`
- 拉取日志：`docker compose -f deployment/spatiotemporal_kriging/docker-compose.yml logs --tail=300`

## 2. 监控告警
- Prometheus：`http://<服务器IP>:9090`
- 告警规则：`deployment/spatiotemporal_kriging/monitoring/alert_rules.yml`
- 建议接入 Alertmanager 并配置企业微信/邮件通知

## 3. 备份恢复
- 手动备份：`deployment/spatiotemporal_kriging/scripts/backup.sh`
- 校验备份：`deployment/spatiotemporal_kriging/scripts/verify_backup.sh <备份目录>`
- 恢复数据：`deployment/spatiotemporal_kriging/scripts/restore.sh <备份目录>`

## 4. 性能优化
- 优先提升 PostgreSQL IOPS 和内存
- 结合访问峰值调整 `WORKFLOW_REDIS_POOL_SIZE`
- 使用 `update.sh` 进行低风险更新，失败自动回滚
