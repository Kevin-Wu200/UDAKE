## 强化学习采样优化概述

### 什么是强化学习采样优化？

强化学习采样优化通过智能体学习最优采样策略，在有限的预算下最大化信息获取量，减少插值不确定性。

### UDAKE强化学习采样优化体系

UDAKE平台提供三种强化学习算法用于采样优化：

#### 1. PPO（近端策略优化）

**核心原理**：
PPO通过重要性采样限制策略更新幅度，提供稳定的策略学习。

**数学模型**：
```
L^CLIP(θ) = 𝔼ₜ[min(rₜ(θ)Âₜ, clip(rₜ(θ), 1-ε, 1+ε)Âₜ)]
```

**优势**：
- 策略学习稳定
- 采样效率高
- 易于调参
- 方差减少效果最佳（+39.8%）

**适用场景**：
- 需要稳定策略学习
- 计算资源充足
- 追求最佳性能

**代码位置**：`/deep_learning/models/sampling_rl/agents.py`

#### 2. DQN（深度Q网络）

**核心原理**：
DQN使用深度神经网络近似Q函数，通过经验回放和目标网络学习最优策略。

**数学模型**：
```
L(θ) = 𝔼[(s,a,r,s')~𝒟][(r + γ maxₐ' Q(s',a';θ⁻) - Q(s,a;θ))²]
```

**优势**：
- 离散动作空间
- 实现简单
- 收敛速度快
- 支持优先经验回放

**适用场景**：
- 离散采样位置
- 需要快速训练
- 动作空间较小

**代码位置**：`/deep_learning/models/sampling_rl/agents.py`

#### 3. A2C（优势演员-评论家）

**核心原理**：
A2C结合策略梯度和价值函数估计，异步训练多个智能体，提高样本效率。

**数学模型**：
```
∇θ J(θ) = 𝔼πθ[∇θ log πθ(a|s)A(s,a)]
```

**优势**：
- 并行训练
- 样本效率高
- 策略评估准确
- 支持连续和离散动作

**适用场景**：
- 需要并行训练
- 动作空间复杂
- 追求高样本效率

**代码位置**：`/deep_learning/models/sampling_rl/agents.py`

## 强化学习参数说明

### PPO参数配置

```python
@dataclass
class PPOConfig:
    learning_rate: float = 0.02
    gamma: float = 0.99              # 折扣因子
    gae_lambda: float = 0.95         # GAE参数
    clip_ratio: float = 0.2          # 裁剪比率
    entropy_coef: float = 0.01       # 熵系数
    value_coef: float = 0.5          # 价值函数系数
    max_grad_norm: float = 0.5       # 梯度裁剪
```

**关键参数说明**：

- `clip_ratio`: 裁剪比率
  - 较小值 (0.1-0.2): 策略更新保守
  - 中等值 (0.2-0.3): 平衡探索和利用
  - 较大值 (0.3-0.5): 策略更新激进
  - 推荐值: 0.2

- `gamma`: 折扣因子
  - 较小值 (0.9-0.95): 关注短期奖励
  - 较大值 (0.95-0.99): 关注长期奖励
  - 推荐值: 0.99

### DQN参数配置

```python
@dataclass
class DQNConfig:
    learning_rate: float = 0.02
    gamma: float = 0.98
    epsilon_start: float = 1.0       # 初始探索率
    epsilon_end: float = 0.01        # 最终探索率
    epsilon_decay: float = 0.995     # 探索率衰减
    replay_buffer_size: int = 10000  # 经验回放缓冲区大小
    batch_size: int = 32
    target_update_freq: int = 100    # 目标网络更新频率
```

### A2C参数配置

```python

class A2CConfig:
    learning_rate_actor: float = 0.02
    learning_rate_critic: float = 0.02
    gamma: float = 0.98
    entropy_coef: float = 0.01
    value_coef: float = 0.5
    n_workers: int = 4               # 工作进程数
```

## 使用示例

### 示例1：PPO采样优化

```python
from deep_learning.models.sampling_rl.agents import PPOAgent
from deep_learning.models.sampling_rl.env import SamplingEnv

# 1. 创建环境
env = SamplingEnv(
    uncertainty_map=uncertainty_map,
    value_map=value_map,
    action_mode='discrete',
    budget=20
)

# 2. 配置PPO参数
config = PPOConfig(
    learning_rate=0.02,
    gamma=0.99,
    clip_ratio=0.2
)

# 3. 创建智能体
agent = PPOAgent(env.observation_space, env.action_space, config)

# 4. 训练
for episode in range(100):
    state = env.reset()
    done = False
    while not done:
        action = agent.select_action(state)
        next_state, reward, done, info = env.step(action)
        agent.update(state, action, reward, next_state, done)
        state = next_state

# 5. 获取采样策略
sampling_points = agent.get_sampling_policy(initial_state, n_points=10)
```

from deep_learning.models.sampling_rl.agents import DQNAgent

## 常见问题解答

### Q1: 如何选择强化学习算法？

**A**: 根据需求选择：

- **PPO**: 追求最佳性能，计算资源充足
- **DQN**: 离散动作空间，需要快速训练
- **A2C**: 需要并行训练，动作空间复杂

### Q2: 如何评估采样策略效果？

**A**: 使用方差减少指标：

```python
# 计算方差减少
variance_reduction = compute_variance_reduction(
    initial_uncertainty,
    final_uncertainty,
    sampling_points
)

print(f"方差减少: {variance_reduction:.4f}")
```

### Q3: 训练不稳定怎么办？

**A**: 使用以下策略：

1. **调整学习率**：降低学习率
2. **增加梯度裁剪**：防止梯度爆炸
3. **增加熵系数**：鼓励探索
4. **使用GAE**：稳定策略梯度

### Q4: 如何加速训练？

**A**: 使用以下策略：

1. **并行训练**：使用A2C或多个worker
2. **GPU加速**：使用GPU计算
3. **经验回放**：提高样本利用率
4. **优先经验回放**：优先学习重要样本

### Q5: 如何处理连续动作空间？

**A**: 使用PPO或SAC（Soft Actor-Critic）：

```python
# 配置PPO支持连续动作
config = PPOConfig(
    continuous_actions=True,
    action_scale=1.0
)
```
