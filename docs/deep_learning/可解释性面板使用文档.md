# 可解释性面板使用文档

## 1. 入口位置
- 模块：`深度学习 -> 模型可解释性增强面板`
- 组件：`apps/frontend/js/components/SpatiotemporalExplainPanel.ts`

## 2. 核心能力
- 支持 `LIME / SHAP / Hybrid` 三种解释方法切换。
- 支持异步任务提交、状态轮询、取消、删除与监控指标查看。
- 支持四个结果视图：`LIME`、`SHAP`、`对比分析`、`时空视图`。
- 任务列表支持增量渲染（滚动/点击“加载更多”）。

## 3. 使用流程
1. 选择模型与解释参数（预测步长、Top-K、重试、批大小、优先级）。
2. 输入 `coords` 与 `series` JSON。
3. 点击“提交解释任务”。
4. 在任务列表中查看进度并打开结果。
5. 在 SHAP 视图中通过阈值与缩放进行交互筛选。

## 4. 快捷键
- `Ctrl/Cmd + Enter`：提交任务
- `Ctrl/Cmd + R`：刷新任务

## 5. 国际化说明
- 组件已接入 `I18n.t`、`I18n.formatDateTime`、`I18n.formatNumber`。
- 新增语言资源文件：
  - `apps/frontend/js/locales/zh-CN.json`
  - `apps/frontend/js/locales/en-US.json`
