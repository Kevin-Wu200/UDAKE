# 空间插值模型训练教程

## 1. 准备

```bash
python3 -m pytest tests/deep_learning/test_spatial_interpolation_models.py -q
```

## 2. 训练模型

```bash
python3 deep_learning/training/train_spatial_interpolation.py --model gnn --epochs 30
python3 deep_learning/training/train_spatial_interpolation.py --model attention --epochs 30
python3 deep_learning/training/train_spatial_interpolation.py --model residual --epochs 30
```

## 3. 评估与推理

```bash
python3 deep_learning/inference/run_spatial_interpolation_inference.py --model gnn --grid-size 20
```

## 4. API 调用

- 训练：`POST /api/dl/spatial/train`
- 推理：`POST /api/dl/spatial/predict`

请求体参考 `docs/deep_learning/API文档.md`。
