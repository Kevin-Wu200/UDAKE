## 融合模型概述

### 什么是模型融合？

模型融合是通过组合多个模型的预测结果，提高预测准确性、稳定性和鲁棒性的技术。

### UDAKE融合模型体系

UDAKE平台提供企业级模型融合系统，支持8种融合策略和多种权重计算方法。

#### 融合策略

1. **简单平均**：等权重平均值
2. **加权平均**：基于性能的加权融合
3. **中位数**：对异常值不敏感
4. **最大最小**：捕捉预测范围
5. **堆叠融合**：通过交叉验证学习元权重
6. **贝叶斯模型平均**：基于贝叶斯框架
7. **方差加权**：根据预测方差的倒数分配权重
8. **动态融合**：根据输入特征动态调整权重

#### 权重计算方法

1. **等权重**：所有模型权重相等
2. **基于RMSE**：根据均方根误差计算权重
3. **基于MAE**：根据平均绝对误差计算权重
4. **基于R²**：根据决定系数计算权重
5. **交叉验证**：通过交叉验证评估模型性能
6. **贝叶斯模型平均**：基于贝叶斯信息准则
7. **基于不确定性**：根据预测不确定性计算权重
8. **自适应**：使用神经网络在线学习权重

## 使用示例

### 示例1：基本融合

```python
from deep_learning.fusion.engine import ModelFusionEngine

# 1. 准备子模型
models = {
    'model1': model1,
    'model2': model2,
    'model3': model3
}

# 2. 创建融合引擎
fusion_engine = ModelFusionEngine(
    strategy="weighted_average",
    weight_method="rmse"
)

# 3. 训练融合模型
fusion_engine.fit(models, X_train, y_train)

# 4. 融合预测
fusion_pred = fusion_engine.predict(X_test)
```

### 示例2：堆叠融合

```python
# 使用堆叠融合
fusion_engine = ModelFusionEngine(
    strategy="stacking",
    weight_method="cross_validation"
)

# 训练
fusion_engine.fit(models, X_train, y_train)

# 预测
fusion_pred = fusion_engine.predict(X_test)
```

### 示例3：动态融合

```python
# 使用动态融合
fusion_engine = ModelFusionEngine(
    strategy="dynamic",
    weight_method="adaptive"
)

# 训练
fusion_engine.fit(models, X_train, y_train)

# 预测（权重会根据输入动态调整）
fusion_pred = fusion_engine.predict(X_test)

# 获取动态权重
dynamic_weights = fusion_engine.get_dynamic_weights(X_test)
```

## 常见问题解答

### Q1: 如何选择融合策略？

**A**: 根据需求选择：

- **简单平均**: 模型性能相近
- **加权平均**: 模型性能差异明显
- **堆叠融合**: 数据充足，追求最优性能
- **贝叶斯模型平均**: 需要概率解释
- **动态融合**: 数据分布多变

### Q2: 如何评估融合效果？

**A**: 使用改进指标：

```python
# 与最佳基线模型对比
best_baseline_rmse = min([m['rmse'] for m in models.values()])
fusion_rmse = evaluate_fusion()

improvement = (best_baseline_rmse - fusion_rmse) / best_baseline_rmse
print(f"改进率: {improvement:.2%}")
```

### Q3: 融合模型过拟合怎么办？

**A**: 使用正则化：

```python
# 增加交叉验证折数
fusion_engine = ModelFusionEngine(n_folds=10)

# 使用正则化权重
fusion_engine.fit(models, X_train, y_train, regularization=True)
```
