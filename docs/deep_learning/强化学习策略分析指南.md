## 策略分析基础

### 什么是策略

在强化学习中，**策略（Policy）**定义了智能体在不同状态下选择动作的规则：

**确定性策略**：π(s) = a，每个状态对应一个确定的动作
**随机策略**：π(a|s) = P(a|s)，每个状态对应动作的概率分布

对于采样优化任务，策略定义了如何在不确定性地图上选择采样点的位置。

### 策略评估指标

#### 1. 基础性能指标

| 指标 | 说明 | 计算方法 | 目标 |
|------|------|----------|------|
| **累积奖励** | 整个episode的总奖励 | Σ R_t | 最大化 |
| **平均奖励** | 平均每步奖励 | mean(R_t) | 最大化 |
| **成功率** | 达到目标的概率 | N_success / N_total | 最大化 |
| **效率** | 达到目标的步数 | steps / budget | 最小化 |

#### 2. 探索-利用指标

| 指标 | 说明 | 计算方法 | 理想值 |
|------|------|----------|--------|
| **策略熵** | 策略的随机性 | -Σ π(a|s) log π(a|s) | 中等（0.3-0.7） |
| **动作覆盖率** | 不同动作被使用的比例 | N_used_actions / N_total_actions | 高（>0.8） |
| **Top1-Top2差距** | 最优和次优动作的Q值差 | Q_max - Q_2nd | 中等（0.1-0.2） |
| **探索率** | 随机动作的比例 | N_random / N_total | 训练初期高 |

#### 3. 采样优化特定指标

| 指标 | 说明 | 计算方法 | 目标 |
|------|------|----------|------|
| **不确定性降低率** | 采样后不确定性的减少 | (U_initial - U_final) / U_initial | 最大化 |
| **空间覆盖率** | 采样区域占总区域的比例 | Area_sampled / Area_total | 最大化 |
| **采样密度** | 单位面积的采样点数 | N_samples / Area_total | 适中 |
| **精度提升率** | 预测精度的提升 | (E_initial - E_final) / E_initial | 最大化 |

### 动作分布分析

#### 1. 动作选择频率

分析智能体对不同动作的偏好，识别过拟合或探索不足的问题。

```python
def analyze_action_frequency(agent, env, n_episodes=10):
    """分析动作选择频率"""
    action_counts = {}
    total_actions = 0

    for _ in range(n_episodes):
        obs = env.reset()
        done = False
        while not done:
            if isinstance(agent, PPOAgent):
                action_idx, _, _ = agent.select_action(obs)
            elif isinstance(agent, DQNAgent):
                action_idx = agent.select_action(obs)

            action_counts[action_idx] = action_counts.get(action_idx, 0) + 1
            total_actions += 1

            action = _action_from_discrete(action_idx, env)
            obs, _, done, _ = env.step(action)

    # 计算统计信息
    action_probs = {k: v / total_actions for k, v in action_counts.items()}
    most_common = max(action_counts.items(), key=lambda x: x[1])
    least_common = min(action_counts.items(), key=lambda x: x[1])

    return {
        "action_counts": action_counts,
        "action_probabilities": action_probs,
        "coverage": len(action_counts) / env.action_space.grid_size,
        "most_common_action": most_common,
        "least_common_action": least_common,
        "gini_coefficient": _calculate_gini(list(action_counts.values()))
    }

def _calculate_gini(values):
    """计算基尼系数"""
    values = sorted(values)
    n = len(values)
    cumulative = np.cumsum(values, dtype=float)
    return (n + 1 - 2 * np.sum(cumulative) / cumulative[-1]) / n
```

#### 2. 动作空间覆盖

评估智能体是否充分利用了动作空间：

**高覆盖率（>0.8）**：探索充分，策略多样化
**中等覆盖率（0.5-0.8）**：部分探索，可能存在偏差
**低覆盖率（<0.5）**：探索不足，容易过拟合

```python
def analyze_value_function(agent, env, grid_resolution=20):
    """分析价值函数在不同状态下的表现"""
    values = []
    positions = []

    # 生成网格状态
    h, w = env.h, env.w
    step_h = max(1, h // grid_resolution)
    step_w = max(1, w // grid_resolution)

    for i in range(0, h, step_h):
        for j in range(0, w, step_w):
            # 构造部分观测
            obs = env._observation()
            obs["sampling_distribution"] = np.zeros_like(obs["sampling_distribution"])
            obs["sampled_values"] = np.zeros_like(obs["sampled_values"])

            value = agent.value(obs)
            values.append(value)
            positions.append((i, j))

    return {
        "values": values,
        "positions": positions,
        "value_stats": {
            "mean": np.mean(values),
            "std": np.std(values),
            "min": np.min(values),
            "max": np.max(values),
            "range": np.max(values) - np.min(values)
        }
    }
```

#### 2. 价值函数平滑性

评估价值函数的平滑性，避免过拟合：

```python
def analyze_value_smoothness(values, positions):
    """分析价值函数的平滑性"""
    if len(values) < 2:
        return {"smoothness": 0.0}

    # 计算相邻状态的价值差异
    values_array = np.array(values)
    positions_array = np.array(positions)

    gradients = []
    for i in range(len(values)):
        for j in range(i + 1, len(values)):
            dist = np.linalg.norm(positions_array[i] - positions_array[j])
            if dist > 0:
                grad = abs(values_array[i] - values_array[j]) / dist
                gradients.append(grad)

    return {
        "mean_gradient": np.mean(gradients),
        "max_gradient": np.max(gradients),
        "smoothness": 1.0 / (1.0 + np.mean(gradients))
    }
```

```python
def track_entropy_over_training(agent, env, n_episodes=50):
    """跟踪训练过程中的策略熵变化"""
    entropy_curve = []

    for episode in range(n_episodes):
        obs = env.reset()
        episode_entropy = []
        done = False

        while not done:
            if isinstance(agent, PPOAgent):
                pred = agent.predict_ppo([obs], deterministic=False)
                episode_entropy.append(pred["policy_entropy"][0])
                action = _action_from_discrete(pred["action_indices"][0], env)
            elif isinstance(agent, DQNAgent):
                # DQN的熵基于Q值分布
                state = flatten_observation(obs)
                q_values = agent._q_values(state)
                probs = _softmax(q_values / agent.epsilon)
                entropy = -np.sum(probs * np.log(probs + 1e-12))
                episode_entropy.append(entropy)
                action = _action_from_discrete(agent.select_action(obs), env)

            obs, _, done, _ = env.step(action)

        if episode_entropy:
            entropy_curve.append(np.mean(episode_entropy))

    return {
        "entropy_curve": entropy_curve,
        "final_entropy": entropy_curve[-1] if entropy_curve else 0.0,
        "entropy_trend": "increasing" if entropy_curve[-1] > entropy_curve[0] else "decreasing",
        "entropy_stability": np.std(entropy_curve[-10:]) if len(entropy_curve) >= 10 else 0.0
    }
```

#### 1. 探索信号跟踪

```python
def analyze_exploration_signals(agent, env, n_episodes=20):
    """分析探索信号"""
    exploration_data = {
        "entropy": [],
        "selected_action_prob": [],
        "max_action_prob": [],
        "top1_top2_gap": [],
        "random_action_ratio": []
    }

    for _ in range(n_episodes):
        obs = env.reset()
        done = False
        episode_random = 0
        episode_total = 0

        while not done:
            if isinstance(agent, PPOAgent):
                pred = agent.predict_ppo([obs], deterministic=False)
                probs = pred["action_probabilities"][0]
                sorted_probs = np.sort(probs)

                exploration_data["entropy"].append(pred["policy_entropy"][0])
                exploration_data["selected_action_prob"].append(pred["selected_action_probabilities"][0])
                exploration_data["max_action_prob"].append(np.max(probs))
                exploration_data["top1_top2_gap"].append(sorted_probs[-1] - sorted_probs[-2])

                action = _action_from_discrete(pred["action_indices"][0], env)
            elif isinstance(agent, DQNAgent):
                action_idx = agent.select_action(obs)
                state = flatten_observation(obs)
                q_values = agent._q_values(state)
                probs = _softmax(q_values / agent.epsilon)
                sorted_probs = np.sort(probs)

                exploration_data["entropy"].append(-np.sum(probs * np.log(probs + 1e-12)))
                exploration_data["selected_action_prob"].append(probs[action_idx])
                exploration_data["max_action_prob"].append(np.max(probs))
                exploration_data["top1_top2_gap"].append(sorted_probs[-1] - sorted_probs[-2])

                # 检查是否是随机动作
                if agent.config.exploration == "epsilon_greedy":
                    episode_random += 1 if agent.rng.random() < agent.epsilon else 0
                    episode_total += 1

                action = _action_from_discrete(action_idx, env)

            obs, _, done, _ = env.step(action)

        if episode_total > 0:
            exploration_data["random_action_ratio"].append(episode_random / episode_total)

    return {
        "exploration_data": exploration_data,
        "summary": {
            "mean_entropy": np.mean(exploration_data["entropy"]),
            "mean_random_ratio": np.mean(exploration_data["random_action_ratio"]) if exploration_data["random_action_ratio"] else 0.0,
            "exploration_trend": "adequate" if np.mean(exploration_data["entropy"]) > 0.3 else "insufficient"
        }
    }
```

#### 2. 动作偏好分析

```python
def analyze_action_preference(agent, env, n_steps=100):
    """分析动作偏好"""
    action_stats = {}
    q_value_stats = {}

    for _ in range(n_steps):
        obs = env.reset()
        if isinstance(agent, PPOAgent):
            pred = agent.predict_ppo([obs], deterministic=True)
            action_idx = pred["action_indices"][0]
            probs = pred["action_probabilities"][0]
            value = pred["state_values"][0]
        elif isinstance(agent, DQNAgent):
            state = flatten_observation(obs)
            q_values = agent._q_values(state)
            action_idx = int(np.argmax(q_values))
            probs = _softmax(q_values / agent.epsilon)
            value = float(np.max(q_values))

        if action_idx not in action_stats:
            action_stats[action_idx] = {
                "count": 0,
                "mean_prob": 0.0,
                "mean_value": 0.0,
                "q_values": []
            }

        action_stats[action_idx]["count"] += 1
        action_stats[action_idx]["mean_prob"] += probs[action_idx]
        action_stats[action_idx]["mean_value"] += value

        if isinstance(agent, DQNAgent):
            action_stats[action_idx]["q_values"].append(q_values[action_idx])

    # 计算统计量
    for action_idx in action_stats:
        stats = action_stats[action_idx]
        stats["mean_prob"] /= stats["count"]
        stats["mean_value"] /= stats["count"]
        if stats["q_values"]:
            stats["q_value_std"] = np.std(stats["q_values"])
            stats["q_value_range"] = np.max(stats["q_values"]) - np.min(stats["q_values"])

    # 排序
    sorted_actions = sorted(action_stats.items(), key=lambda x: x[1]["count"], reverse=True)

    return {
        "action_stats": action_stats,
        "top_actions": sorted_actions[:5],
        "action_diversity": len(action_stats),
        "preference_concentration": sorted_actions[0][1]["count"] / n_steps if sorted_actions else 0.0
    }
```

### 离线评估

离线评估使用已收集的数据评估策略，不需要与环境交互。

#### 1. 历史轨迹评估

```python
def evaluate_offline(agent, trajectories):
    """使用历史轨迹评估策略"""
    total_reward = 0.0
    correct_actions = 0
    total_actions = 0

    for trajectory in trajectories:
        for transition in trajectory:
            obs = transition["observation"]
            true_action = transition["action"]

            # 智能体选择动作
            if isinstance(agent, PPOAgent):
                pred = agent.predict_ppo([obs], deterministic=True)
                predicted_action = pred["action_indices"][0]
            elif isinstance(agent, DQNAgent):
                predicted_action = agent.select_action(obs, deterministic=True)

            # 评估
            total_reward += transition["reward"]
            if predicted_action == true_action:
                correct_actions += 1
            total_actions += 1

    return {
        "total_reward": total_reward,
        "accuracy": correct_actions / total_actions if total_actions > 0 else 0.0,
        "mean_reward_per_step": total_reward / total_actions if total_actions > 0 else 0.0
    }
```

#### 2. 重要性采样评估

```python
def evaluate_importance_sampling(agent, trajectories, behavior_policy, discount=0.99):
    """使用重要性采样评估策略"""
    episode_returns = []

    for trajectory in trajectories:
        rho = 1.0  # 重要性权重
        weighted_return = 0.0
        G = 0.0

        for t, transition in enumerate(reversed(trajectory)):
            obs = transition["observation"]
            action = transition["action"]
            reward = transition["reward"]

            # 计算重要性权重
            if isinstance(agent, PPOAgent):
                pred = agent.predict_ppo([obs], deterministic=False)
                target_prob = pred["action_probabilities"][0][action]
            elif isinstance(agent, DQNAgent):
                state = flatten_observation(obs)
                q_values = agent._q_values(state)
                probs = _softmax(q_values / agent.epsilon)
                target_prob = probs[action]

            behavior_prob = behavior_policy(obs, action)
            rho *= target_prob / (behavior_prob + 1e-12)

            # 限制权重避免方差过大
            rho = min(rho, 10.0)

            G = reward + discount * G
            weighted_return += rho * G

        episode_returns.append(weighted_return)

    return {
        "mean_return": np.mean(episode_returns),
        "std_return": np.std(episode_returns),
        "confidence_interval": (
            np.mean(episode_returns) - 1.96 * np.std(episode_returns) / np.sqrt(len(episode_returns)),
            np.mean(episode_returns) + 1.96 * np.std(episode_returns) / np.sqrt(len(episode_returns))
        )
    }
```

### 在线评估

在线评估通过实际与环境交互评估策略。

#### 1. 标准评估

```python
def evaluate_online(agent, env, n_episodes=10, render=False):
    """在线评估策略"""
    episode_rewards = []
    episode_lengths = []
    success_count = 0

    for episode in range(n_episodes):
        obs = env.reset()
        episode_reward = 0.0
        step = 0
        done = False

        while not done:
            # 选择动作
            if isinstance(agent, PPOAgent):
                action_idx, _, _ = agent.select_action(obs, deterministic=True)
            elif isinstance(agent, DQNAgent):
                action_idx = agent.select_action(obs, deterministic=True)

            action = _action_from_discrete(action_idx, env)

            # 执行动作
            obs, reward, done, info = env.step(action)
            episode_reward += reward
            step += 1

            if render:
                env.render()

        episode_rewards.append(episode_reward)
        episode_lengths.append(step)

        if info.get("termination_reason") == "target_accuracy":
            success_count += 1

    return {
        "mean_reward": np.mean(episode_rewards),
        "std_reward": np.std(episode_rewards),
        "mean_length": np.mean(episode_lengths),
        "success_rate": success_count / n_episodes,
        "reward_distribution": {
            "min": np.min(episode_rewards),
            "max": np.max(episode_rewards),
            "median": np.median(episode_rewards)
        }
    }
```

```python
def cross_validate_evaluate(agent_class, env_config, train_configs, n_folds=5):
    """交叉验证评估"""
    fold_results = []

    for fold in range(n_folds):
        # 创建环境
        env = SamplingEnv(**env_config, seed=fold)

        # 创建智能体
        agent = agent_class(**train_configs[fold % len(train_configs)])

        # 训练
        train_result = train_agent(env, agent, episodes=50)

        # 评估
        eval_result = evaluate_online(agent, env, n_episodes=10)

        fold_results.append({
            "fold": fold,
            "train_result": train_result,
            "eval_result": eval_result
        })

    # 汇总结果
    mean_rewards = [r["eval_result"]["mean_reward"] for r in fold_results]
    success_rates = [r["eval_result"]["success_rate"] for r in fold_results]

    return {
        "fold_results": fold_results,
        "summary": {
            "mean_reward_mean": np.mean(mean_rewards),
            "mean_reward_std": np.std(mean_rewards),
            "success_rate_mean": np.mean(success_rates),
            "success_rate_std": np.std(success_rates)
        }
    }
```

```python
def compare_algorithms(algorithms, env, n_episodes=20):
```

```python
def analyze_temporal_consistency(agent, env, trajectory_length=20):
    """分析时间一致性"""
    obs = env.reset()
    trajectory = []

    for _ in range(trajectory_length):
        # 获取动作概率分布
        if isinstance(agent, PPOAgent):
            pred = agent.predict_ppo([obs], deterministic=False)
            probs = pred["action_probabilities"][0]
        elif isinstance(agent, DQNAgent):
            state = flatten_observation(obs)
            q_values = agent._q_values(state)
            probs = _softmax(q_values / agent.epsilon)

        trajectory.append({
            "step": len(trajectory),
            "probs": probs.copy(),
            "max_prob": np.max(probs),
            "entropy": -np.sum(probs * np.log(probs + 1e-12))
        })

        # 执行动作
        if isinstance(agent, PPOAgent):
            action_idx = pred["action_indices"][0]
        elif isinstance(agent, DQNAgent):
            action_idx = agent.select_action(obs, deterministic=True)

        action = _action_from_discrete(action_idx, env)
        obs, _, done, _ = env.step(action)

        if done:
            break

    kl_divergences = []
    for i in range(len(trajectory) - 1):
        p = trajectory[i]["probs"] + 1e-12
        q = trajectory[i + 1]["probs"] + 1e-12
        kl = np.sum(p * np.log(p / q))
        kl_divergences.append(kl)

    return {
        "mean_kl_divergence": np.mean(kl_divergences),
        "max_kl_divergence": np.max(kl_divergences),
        "consistency_level": "stable" if np.mean(kl_divergences) < 0.1 else "unstable",
        "trajectory": trajectory
    }
```

```python
class RewardShaper:
    """奖励塑形器"""

    def __init__(self, potential_function=None, gamma=0.99):
        self.potential_function = potential_function
        self.gamma = gamma

    def shape_reward(self, reward, state, next_state):
        """应用奖励塑形"""
        if self.potential_function is None:
            return reward

        phi_t = self.potential_function(state)
        phi_t1 = self.potential_function(next_state)

        shaped_reward = reward + self.gamma * phi_t1 - phi_t
        return shaped_reward
```

```python
def analyze_policy_with_lime(agent, env, observation):

        original_prob = probs[original_action]

    perturbed_obs = []
    actions = []
    probs = []

    for _ in range(n_samples):
        # 扰动观测
        perturbed = {
            k: v + np.random.normal(0, 0.1 * np.std(v), v.shape)
            for k, v in observation.items()
        }

        # 获取动作
        if isinstance(agent, PPOAgent):
```

```python
def analyze_policy_with_shap(agent, env, observation, background_samples=None):
    """使用SHAP分析策略"""
    try:
        import shap
    except ImportError:
        return {"error": "SHAP not installed"}

    # 准备背景数据
    if background_samples is None:
        background_samples = [env._observation() for _ in range(10)]

    # 定义预测函数
    def predict_fn(X):
        results = []
        for x in X:
            obs = {
                "sampling_distribution": x[:env.h * env.w].reshape(env.h, env.w),
                "uncertainty_map": x[env.h * env.w:2 * env.h * env.w].reshape(env.h, env.w),
                "sampled_values": x[2 * env.h * env.w:3 * env.h * env.w].reshape(env.h, env.w),
                "spatial_features": x[3 * env.h * env.w:3 * env.h * env.w + 4 * env.h * env.w].reshape(4, env.h, env.w),
                "boundary_info": x[3 * env.h * env.w + 4 * env.h * env.w:]
            }

            if isinstance(agent, PPOAgent):
                pred = agent.predict_ppo([obs], deterministic=True)
                results.append(pred["state_values"][0])
            elif isinstance(agent, DQNAgent):
                state = flatten_observation(obs)
                q_values = agent._q_values(state)
                results.append(np.max(q_values))

        return np.array(results)

    # 创建SHAP解释器
    background = np.array([flatten_observation(obs) for obs in background_samples])
    explainer = shap.KernelExplainer(predict_fn, background)

    # 计算SHAP值
    obs_flat = flatten_observation(observation)
    shap_values = explainer.shap_values(obs_flat)

    return {
        "shap_values": shap_values,
        "base_value": explainer.expected_value,
        "feature_importance": np.abs(shap_values).mean(axis=0)
```

```python
                obs, reward, done, info = env.step(action)
                episode_reward += reward

                # 更新噪声观测
                noisy_obs = {
                    k: v + np.random.normal(0, noise_level * np.std(v), v.shape)
                    for k, v in obs.items()
                }

            total_reward += episode_reward
            if info.get("termination_reason") == "target_accuracy":
                success_count += 1

        results[noise_level] = {
            "mean_reward": total_reward / n_episodes,
            "success_rate": success_count / n_episodes
        }

    return results
```

```python
    rewards = [r["mean_reward"] for r in results]
    success_rates = [r["success_rate"] for r in results]

    return {

        "reward_variance": np.var(rewards),
        "reward_range": (np.min(rewards), np.max(rewards)),
```

```python
def analyze_zero_shot_transfer(agent, source_env, target_envs, n_episodes=5):
    """分析零样本迁移"""
    transfer_results = []

    # 在源环境评估
    source_eval = evaluate_online(agent, source_env, n_episodes=n_episodes)

    for target_name, target_env in target_envs.items():
        # 在目标环境评估
        target_eval = evaluate_online(agent, target_env, n_episodes=n_episodes)

        # 计算迁移性能
        transfer_ratio = target_eval["mean_reward"] / source_eval["mean_reward"]

        transfer_results.append({
            "target_env": target_name,
            "source_reward": source_eval["mean_reward"],
            "target_reward": target_eval["mean_reward"],
            "transfer_ratio": transfer_ratio,
            "transfer_success": transfer_ratio > 0.8
        })

    return {
        "source_performance": source_eval,
        "transfer_results": transfer_results,
        "mean_transfer_ratio": np.mean([r["transfer_ratio"] for r in transfer_results])
    }
```

## 最佳实践

### 1. 分析流程

**完整的策略分析流程**：

1. **基础评估**：计算基本性能指标
2. **可视化分析**：绘制动作分布、熵变化等
3. **深度诊断**：识别具体问题（探索不足、过拟合等）
4. **优化调整**：根据诊断结果调整参数
5. **验证效果**：重新评估验证改进效果

### 2. 常见模式

**探索不足的处理**：
- 增加熵系数
- 使用更温和的探索策略
- 增加初始探索率
- 减慢探索率衰减

**过拟合的处理**：
- 增加正则化
- 减少模型复杂度
- 增加数据多样性
- 使用早停

**收敛困难的处理**：
- 降低学习率
- 增加批量大小
- 使用更稳定的优化器
- 增加梯度裁剪
