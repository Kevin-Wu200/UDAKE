# 深度学习基础架构设计文档

## 目标

构建可扩展的深度学习底座，支持空间插值、异常检测、不确定性建模、强化学习采样和时空模型。

## 分层架构

1. `deep_learning/utils`：通用工具（设备管理、预处理、缓存、监控、日志）。
2. `deep_learning/training`：训练引擎（Trainer 抽象、Lightning 风格实现）。
3. `deep_learning/models`：模型注册、版本、导出与量化。
4. `deep_learning/inference`：批量/流式/异步预测。
5. `deep_learning/config`：YAML 配置模板与加载校验。
6. `services/backend/app/dl_services`：FastAPI 集成层。

## 可扩展策略

- 通过 `ModelRegistry.register` 动态注册新模型。
- 通过 `ConfigLoader` 处理环境差异与多环境配置。
- 通过 `MetricMonitor` + `DashboardBuilder` 暴露统一监控视图。
