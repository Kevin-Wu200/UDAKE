# 深度学习基础架构设计文档

## 目标

构建可扩展的深度学习底座，支持空间插值、异常检测、不确定性建模、强化学习采样和时空模型。

## 分层架构

1. `deep_learning/utils`：通用工具（设备管理、预处理、缓存、监控、日志）。
2. `deep_learning/training`：训练引擎（Trainer 抽象、Lightning 风格实现）。
3. `deep_learning/models`：模型注册、版本、导出与量化。
4. `deep_learning/inference`：批量/流式/异步预测。
5. `deep_learning/config`：YAML 配置模板与加载校验。
6. `services/backend/app/dl_services`：FastAPI 集成层。

## 阶段2空间插值架构

1. 图神经网络克里金（`GNNKrigingModel`）
- `SpatialGraphBuilder` 支持 KNN/半径/Voronoi/Delaunay 图。
- `GCN + GAT + EdgeConv` 并行特征编码。
- 融合传统克里金先验均值/方差，采用残差学习。

2. 注意力克里金（`AttentionKrigingModel`）
- Transformer 编码器（多头注意力、前馈网络、层归一化、残差连接）。
- 采样点与查询点双向交叉注意力。
- 动态距离感知与方差感知加权。

3. 残差克里金（`ResidualKrigingModel`）
- 传统克里金基线（普通/通用）+ 残差网络。
- MLP/CNN/Hybrid 三分支可选。
- ResNet 风格跳连与 Dense 风格混合路径。

## 可扩展策略

- 通过 `ModelRegistry.register` 动态注册新模型。
- 通过 `ConfigLoader` 处理环境差异与多环境配置。
- 通过 `MetricMonitor` + `DashboardBuilder` 暴露统一监控视图。
- 通过 `SpatialInterpolationIntegrator` 接入实时插值系统、缓存和事件总线。

## 阶段6时空预测架构

1. `SpatioTemporalTransformer`
- 空间编码器：`GCN + GAT + 空间多头注意力 + 空间位置编码`。
- 时间编码器：`GRU/LSTM-like + 时间Transformer + 时间注意力 + 时间位置编码`。
- 时空融合：`ST-Attention + Cross-Attention + Concat/Add/Gating + Residual`。
- 预测头：多步预测均值与方差输出。

2. 经典时空模型
- `GCNLSTMModel`：图卷积提取空间特征 + 多层/双向LSTM + 门控耦合。
- `ConvLSTMModel`：局部/全局多尺度卷积 + 递归记忆 + 空间/时间注意力。
- `STGCNModel`：空间图卷积 + 时间膨胀卷积 + 多层ST-Block残差堆叠。

3. 数据与训练推理闭环
- `deep_learning/models/spatiotemporal/data.py`：时空数据集、增强、特征工程、标准化、图构建、滑窗加载。
- `deep_learning/models/spatiotemporal/training.py`：序列长度/预测步长配置、余弦退火+预热、早停、梯度裁剪代理。
- `deep_learning/inference/spatiotemporal_inference.py`：批量预测、在线预测与增量更新。
- `deep_learning/models/spatiotemporal/evaluation.py`：RMSE/MAE/MAPE/R²/sMAPE/CRPS，时间维与空间维评估、报告生成。
