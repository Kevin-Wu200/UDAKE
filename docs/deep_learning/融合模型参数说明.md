# 融合模型参数说明文档

## 概述

融合模型系统提供了丰富的配置参数，用于控制融合策略、权重计算、自适应学习等各个方面。本文档详细介绍了所有参数的含义、取值范围、默认值和使用建议，帮助用户正确配置和使用融合模型。

## 参数总览

### FusionConfig 参数

| 参数名称 | 类型 | 默认值 | 取值范围 | 必需 | 说明 |
|---------|------|--------|---------|------|------|
| `strategy` | FusionStrategy | `weighted_average` | 见策略枚举 | 否 | 融合策略 |
| `weight_method` | WeightMethod | `rmse_based` | 见权重方法枚举 | 否 | 权重计算方法 |
| `adaptive_mode` | AdaptiveLearningMode | `neural` | 见自适应模式枚举 | 否 | 自适应学习模式 |
| `min_weight` | float | `0.0` | [0.0, 1.0] | 否 | 最小权重 |
| `max_weight` | float | `1.0` | [0.0, 1.0] | 否 | 最大权重 |
| `normalize` | bool | `True` | - | 否 | 是否归一化权重 |
| `smoothing` | bool | `False` | - | 否 | 是否平滑权重 |
| `smoothing_factor` | float | `0.1` | (0.0, 1.0) | 否 | 平滑因子 |
| `n_folds` | int | `5` | [2, 10] | 否 | 交叉验证折数 |
| `enable_uncertainty` | bool | `True` | - | 否 | 是否启用不确定性估计 |

## 参数详解

### 1. 融合策略（strategy）

#### 1.1 参数说明
- **类型**：`FusionStrategy` (枚举)
- **默认值**：`weighted_average`
- **必需**：否
- **影响**：决定如何组合多个模型的预测结果

#### 1.2 可选值

| 值 | 说明 | 需要真值 | 计算复杂度 | 适用场景 |
|----|------|---------|-----------|---------|
| `simple_average` | 简单平均 | 否 | 低 | 模型性能相近 |
| `weighted_average` | 加权平均 | 否 | 低 | 模型性能差异明显 |
| `median` | 中位数 | 否 | 中 | 存在异常预测 |
| `max_min` | 最大最小 | 否 | 低 | 需要保守估计 |
| `stacking` | 堆叠融合 | 是 | 高 | 数据充足，精度要求高 |
| `bayesian_model_average` | 贝叶斯模型平均 | 否 | 中 | 需要概率解释 |
| `variance_weighted` | 方差加权 | 否 | 中 | 有方差估计 |
| `dynamic` | 动态融合 | 否 | 高 | 数据分布多变 |

#### 1.3 使用建议

**选择 `simple_average`**，如果：
- 模型性能相近且无明显差异
- 需要快速实现和部署
- 缺乏标注数据用于性能评估
- 对计算效率要求高

**选择 `weighted_average`**，如果：
- 模型性能差异明显
- 有可靠的性能评估数据
- 需要明确模型贡献度
- 对融合精度要求较高

**选择 `median`**，如果：
- 存在异常预测或离群值
- 模型数量较多（≥ 5）
- 需要高鲁棒性
- 数据质量不稳定

**选择 `stacking`**，如果：
- 有充足的标注数据（> 1000样本）
- 对融合精度要求极高
- 可以接受离线训练
- 模型间互补性强

**选择 `dynamic`**，如果：
- 数据分布多变
- 有上下文信息可用
- 需要自适应能力
- 不同区域需要不同策略

#### 1.4 示例代码

```python
from deep_learning.fusion import FusionConfig, FusionStrategy

# 使用简单平均
config = FusionConfig(strategy=FusionStrategy.SIMPLE_AVERAGE)

# 使用加权平均
config = FusionConfig(strategy=FusionStrategy.WEIGHTED_AVERAGE)

# 使用动态融合
config = FusionConfig(strategy=FusionStrategy.DYNAMIC)
```

### 2. 权重计算方法（weight_method）

#### 2.1 参数说明
- **类型**：`WeightMethod` (枚举)
- **默认值**：`rmse_based`
- **必需**：否
- **影响**：决定如何计算各模型的权重

#### 2.2 可选值

| 值 | 说明 | 需要真值 | 计算公式 | 适用场景 |
|----|------|---------|---------|---------|
| `equal` | 等权重 | 否 | w_i = 1/N | 简单平均策略 |
| `rmse_based` | 基于RMSE | 是 | w_i ∝ 1/RMSE_i² | 通用场景 |
| `mae_based` | 基于MAE | 是 | w_i ∝ 1/MAE_i | 对异常值敏感 |
| `r2_based` | 基于R² | 是 | w_i ∝ R2_i | 需要高相关性 |
| `cross_validation` | 交叉验证 | 是 | CV评估 | 数据充足 |
| `bma` | 贝叶斯模型平均 | 否 | w_i ∝ exp(-BIC/2) | 概率解释 |
| `uncertainty_based` | 基于不确定性 | 否 | w_i ∝ 1/σ_i² | 有方差估计 |
| `adaptive` | 自适应 | 是 | w = f(metrics) | 在线学习 |

#### 2.3 使用建议

**选择 `rmse_based`**，如果：
- 需要惩罚大误差
- 对预测精度要求高
- 数据分布相对均匀
- 通用场景

**选择 `mae_based`**，如果：
- 对异常值不敏感
- 数据中存在噪声
- 需要稳定的权重计算
- 数据分布偏斜

**选择 `r2_based`**，如果：
- 需要高相关性模型
- 模型性能差异主要体现在相关性上
- 需要解释模型拟合程度

**选择 `uncertainty_based`**，如果：
- 模型提供可靠的方差估计
- 不确定性量化重要
- 无标注数据
- 需要动态权重

**选择 `adaptive`**，如果：
- 数据分布会变化
- 需要在线学习
- 有充足的新数据
- 对自适应能力要求高

#### 2.4 示例代码

```python
from deep_learning.fusion import FusionConfig, WeightMethod

# 基于RMSE
config = FusionConfig(weight_method=WeightMethod.RMSE_BASED)

# 基于不确定性
config = FusionConfig(weight_method=WeightMethod.UNCERTAINTY_BASED)

# 自适应学习
config = FusionConfig(weight_method=WeightMethod.ADAPTIVE)
```

### 3. 自适应学习模式（adaptive_mode）

#### 3.1 参数说明
- **类型**：`AdaptiveLearningMode` (枚举)
- **默认值**：`neural`
- **必需**：否
- **影响**：控制自适应权重的学习方式

#### 3.2 可选值

| 值 | 说明 | 特点 | 适用场景 |
|----|------|------|---------|
| `neural` | 神经网络 | 捕捉复杂非线性关系 | 大数据，复杂模式 |
| `attention` | 注意力机制 | 关注重要特征 | 时序数据，序列预测 |

#### 3.3 使用建议

**选择 `neural`**，如果：
- 有充足训练数据（> 1000样本）
- 特征维度较高
- 权重与特征关系复杂
- 需要捕捉非线性模式

**选择 `attention`**，如果：
- 处理时序数据
- 需要关注特定时刻
- 特征有隐含结构
- 需要可解释性

#### 3.4 示例代码

```python
from deep_learning.fusion import FusionConfig, AdaptiveLearningMode

# 神经网络模式
config = FusionConfig(
    adaptive_mode=AdaptiveLearningMode.NEURAL
)

# 注意力模式
config = FusionConfig(
    adaptive_mode=AdaptiveLearningMode.ATTENTION
)
```

### 4. 最小权重（min_weight）

#### 4.1 参数说明
- **类型**：`float`
- **默认值**：`0.0`
- **必需**：否
- **影响**：限制模型的最小权重

#### 4.2 取值范围
- **最小值**：`0.0`（允许权重为0）
- **最大值**：`1.0`（所有模型权重相等）
- **推荐值**：`0.0` - `0.1`

#### 4.3 使用建议

**设置较高值（≥ 0.05）**，如果：
- 希望所有模型都参与融合
- 模型数量较少（< 5）
- 需要保留模型多样性
- 对某些模型信心不足

**设置为 0**，如果：
- 允许剔除低性能模型
- 模型数量较多（≥ 10）
- 需要突出高性能模型
- 对精度要求高

#### 4.4 示例代码

```python
from deep_learning.fusion import FusionConfig

# 允许权重为0
config = FusionConfig(min_weight=0.0)

# 设置最小权重为0.05
config = FusionConfig(min_weight=0.05)
```

### 5. 最大权重（max_weight）

#### 5.1 参数说明
- **类型**：`float`
- **默认值**：`1.0`
- **必需**：否
- **影响**：限制模型的最大权重

#### 5.2 取值范围
- **最小值**：`0.0`（无意义）
- **最大值**：`1.0`（允许单一模型主导）
- **推荐值**：`0.5` - `0.8`

#### 5.3 使用建议

**设置较低值（≤ 0.5）**，如果：
- 需要平衡融合
- 避免单一模型主导
- 希望保持模型多样性
- 对稳定性要求高

**设置较高值（≥ 0.7）**，如果：
- 有明显最优模型
- 需要突出高性能模型
- 对精度要求高
- 模型性能差异大

**设置为 1.0**，如果：
- 允许单一模型主导
- 模型性能差异极大
- 需要最大化利用最优模型

#### 5.4 示例代码

```python
from deep_learning.fusion import FusionConfig

# 限制最大权重为0.7
config = FusionConfig(max_weight=0.7)

# 允许单一模型主导
config = FusionConfig(max_weight=1.0)
```

### 6. 归一化权重（normalize）

#### 6.1 参数说明
- **类型**：`bool`
- **默认值**：`True`
- **必需**：否
- **影响**：是否将权重归一化为概率分布

#### 6.2 使用建议

**设置为 `True`**，如果：
- 需要权重和为1
- 需要概率解释
- 使用概率融合策略
- 通用场景

**设置为 `False`**，如果：
- 使用原始权重分数
- 不需要概率解释
- 权重用于其他目的

#### 6.3 示例代码

```python
from deep_learning.fusion import FusionConfig

# 启用归一化
config = FusionConfig(normalize=True)

# 禁用归一化
config = FusionConfig(normalize=False)
```

### 7. 平滑权重（smoothing）

#### 7.1 参数说明
- **类型**：`bool`
- **默认值**：`False`
- **必需**：否
- **影响**：是否对权重进行平滑处理

#### 7.2 使用建议

**设置为 `True`**，如果：
- 权重波动较大
- 需要稳定的权重
- 在线学习场景
- 数据有噪声

**设置为 `False`**，如果：
- 需要精确的权重
- 权重已经稳定
- 离线训练场景
- 数据质量高

#### 7.3 示例代码

```python
from deep_learning.fusion import FusionConfig

# 启用平滑
config = FusionConfig(
    smoothing=True,
    smoothing_factor=0.1
)
```

### 8. 平滑因子（smoothing_factor）

#### 8.1 参数说明
- **类型**：`float`
- **默认值**：`0.1`
- **必需**：否
- **影响**：控制平滑的强度

#### 8.2 取值范围
- **最小值**：`0.0`（无平滑）
- **最大值**：`1.0`（最大平滑）
- **推荐值**：`0.05` - `0.2`

#### 8.3 使用建议

**设置较低值（≤ 0.05）**，如果：
- 需要轻微平滑
- 权重波动较小
- 保持权重精度

**设置中等值（0.05 - 0.2）**，如果：
- 需要适度平滑
- 权重波动中等
- 平衡稳定性和精度

**设置较高值（≥ 0.2）**，如果：
- 需要强烈平滑
- 权重波动较大
- 优先考虑稳定性

#### 8.4 示例代码

```python
from deep_learning.fusion import FusionConfig

# 轻微平滑
config = FusionConfig(
    smoothing=True,
    smoothing_factor=0.05
)

# 适度平滑
config = FusionConfig(
    smoothing=True,
    smoothing_factor=0.1
)

# 强烈平滑
config = FusionConfig(
    smoothing=True,
    smoothing_factor=0.2
)
```

### 9. 交叉验证折数（n_folds）

#### 9.1 参数说明
- **类型**：`int`
- **默认值**：`5`
- **必需**：否
- **影响**：交叉验证的折数，用于堆叠融合等策略

#### 9.2 取值范围
- **最小值**：`2`（最少折数）
- **最大值**：`10`（推荐最大值）
- **推荐值**：`3` - `5`

#### 9.3 使用建议

**设置为 3**，如果：
- 样本量较小（< 500）
- 需要快速训练
- 计算资源有限

**设置为 5**，如果：
- 样本量中等（500 - 2000）
- 平衡训练时间和评估准确性
- 通用场景

**设置为 10**，如果：
- 样本量充足（> 2000）
- 需要准确评估
- 计算资源充足

#### 9.4 示例代码

```python
from deep_learning.fusion import FusionConfig

# 3折交叉验证
config = FusionConfig(n_folds=3)

# 5折交叉验证
config = FusionConfig(n_folds=5)

# 10折交叉验证
config = FusionConfig(n_folds=10)
```

### 10. 启用不确定性估计（enable_uncertainty）

#### 10.1 参数说明
- **类型**：`bool`
- **默认值**：`True`
- **必需**：否
- **影响**：是否计算并返回融合结果的不确定性估计

#### 10.2 使用建议

**设置为 `True`**，如果：
- 需要量化预测风险
- 需要不确定性信息辅助决策
- 用于风险评估或主动学习
- 需要评估预测可靠性

**设置为 `False`**，如果：
- 只需要点预测
- 不关心不确定性
- 减少计算开销
- 模型不提供方差估计

#### 10.3 示例代码

```python
from deep_learning.fusion import FusionConfig

# 启用不确定性估计
config = FusionConfig(enable_uncertainty=True)

# 禁用不确定性估计
config = FusionConfig(enable_uncertainty=False)
```

## 参数组合建议

### 1. 保守配置
适用于需要高稳定性和低风险的场景：

```python
from deep_learning.fusion import FusionConfig, FusionStrategy, WeightMethod

config = FusionConfig(
    strategy=FusionStrategy.MEDIAN,
    weight_method=WeightMethod.EQUAL,
    min_weight=0.05,
    max_weight=0.5,
    smoothing=True,
    smoothing_factor=0.15,
    enable_uncertainty=True
)
```

### 2. 精度优先配置
适用于对精度要求极高的场景：

```python
from deep_learning.fusion import FusionConfig, FusionStrategy, WeightMethod

config = FusionConfig(
    strategy=FusionStrategy.STACKING,
    weight_method=WeightMethod.RMSE_BASED,
    min_weight=0.0,
    max_weight=1.0,
    n_folds=5,
    enable_uncertainty=True
)
```

### 3. 平衡配置
适用于通用场景，平衡精度和稳定性：

```python
from deep_learning.fusion import FusionConfig, FusionStrategy, WeightMethod

config = FusionConfig(
    strategy=FusionStrategy.WEIGHTED_AVERAGE,
    weight_method=WeightMethod.RMSE_BASED,
    min_weight=0.0,
    max_weight=0.7,
    smoothing=False,
    enable_uncertainty=True
)
```

### 4. 自适应配置
适用于数据分布多变的场景：

```python
from deep_learning.fusion import FusionConfig, FusionStrategy, WeightMethod, AdaptiveLearningMode

config = FusionConfig(
    strategy=FusionStrategy.DYNAMIC,
    weight_method=WeightMethod.ADAPTIVE,
    adaptive_mode=AdaptiveLearningMode.NEURAL,
    smoothing=True,
    smoothing_factor=0.1,
    enable_uncertainty=True
)
```

### 5. 快速配置
适用于需要快速实现的场景：

```python
from deep_learning.fusion import FusionConfig, FusionStrategy

config = FusionConfig(
    strategy=FusionStrategy.SIMPLE_AVERAGE,
    enable_uncertainty=False
)
```

## 参数调优指南

### 1. 网格搜索
系统性尝试参数组合：

```python
from deep_learning.fusion import FusionConfig, FusionStrategy, WeightMethod

strategies = [
    FusionStrategy.WEIGHTED_AVERAGE,
    FusionStrategy.STACKING,
    FusionStrategy.DYNAMIC
]

weight_methods = [
    WeightMethod.RMSE_BASED,
    WeightMethod.MAE_BASED,
    WeightMethod.ADAPTIVE
]

for strategy in strategies:
    for weight_method in weight_methods:
        config = FusionConfig(
            strategy=strategy,
            weight_method=weight_method
        )
        result = engine.fuse(models, config=config, true_values=true_values)
        print(f"{strategy.value} + {weight_method.value}: RMSE={result.metrics['rmse']:.4f}")
```

### 2. 对比分析
比较不同参数配置的效果：

```python
configs = [
    FusionConfig(strategy=FusionStrategy.WEIGHTED_AVERAGE),
    FusionConfig(strategy=FusionStrategy.STACKING, n_folds=5),
    FusionConfig(strategy=FusionStrategy.DYNAMIC)
]

results = engine.compare_strategies(
    models=models,
    base_config=FusionConfig(),
    true_values=true_values
)

for strategy, result in results.items():
    print(f"{strategy}:")
    print(f"  RMSE: {result.metrics['rmse']:.4f}")
    print(f"  MAE: {result.metrics['mae']:.4f}")
    print(f"  R²: {result.metrics['r2']:.4f}")
```

### 3. 优化权重方法
寻找最优权重计算方法：

```python
result = engine.optimize_weight_methods(
    models=models,
    base_config=FusionConfig(),
    true_values=true_values

print("权重方法优化结果:")

    print(f"  {method}:")
    print(f"    RMSE: {scores['rmse']:.4f}")
    print(f"    MAE: {scores['mae']:.4f}")
    print(f"    R²: {scores['r2']:.4f}")

```

## 常见问题

### Q1: 如何选择合适的融合策略？
**答案**：根据以下因素选择：
- 数据量：数据充足用堆叠，数据不足用简单策略
- 性能差异：差异大用加权，差异小用平均
- 需求：精度高用堆叠，稳定性高用中位数
- 计算资源：资源有限用简单策略

### Q2: 权重约束如何设置？
**答案**：
- `min_weight`：通常设置为0，允许剔除低性能模型
- `max_weight`：通常设置为0.5-0.7，避免单一模型主导
- 特殊场景：需要多样性时提高min_weight，需要精度时提高max_weight

### Q3: 是否需要启用平滑？
**答案**：
- 在线学习：建议启用，减少权重波动
- 离线训练：可选，根据数据质量决定
- 数据有噪声：建议启用
- 数据质量高：可选

### Q4: 交叉验证折数如何选择？
**答案**：
- 样本量 < 500：3折
- 样本量 500-2000：5折
- 样本量 > 2000：10折
- 计算资源有限：减少折数

### Q5: 不确定性估计何时使用？
**答案**：
- 风险管理：需要量化预测风险
- 决策支持：需要不确定性信息
- 模型评估：评估预测可靠性
- 主动学习：基于不确定性选择样本
