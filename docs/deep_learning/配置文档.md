# 配置文档

## 配置文件

- `deep_learning/config/model_config.yaml`
- `deep_learning/config/training_config.yaml`
- `deep_learning/config/inference_config.yaml`

## 继承机制

通过 `_base: xxx.yaml` 继承基配置，再覆盖差异字段。

## 环境变量覆盖

约定前缀：`DL_`

示例：

- `DL_TRAINING__MAX_EPOCHS=120`
- `DL_INFERENCE__BATCH_SIZE=256`

双下划线 `__` 用于表示嵌套路径。
