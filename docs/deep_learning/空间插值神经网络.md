# 阶段2：空间插值神经网络说明

## 模块总览

目录：`deep_learning/models/spatial_interpolation`

包含三类模型：
- `GNNKrigingModel`：图构建 + GCN/GAT/EdgeConv + 多尺度注意力 + 克里金残差学习。
- `AttentionKrigingModel`：Transformer 编码器 + 双向交叉注意力 + 距离/方差感知动态权重。
- `ResidualKrigingModel`：普通/通用克里金基线 + MLP/CNN/Hybrid 残差分支 + 多尺度融合。

## 关键能力

1. 图构建与边权重
- 距离图、KNN 图、Voronoi 图、Delaunay 图
- 距离权重、协方差权重、混合权重

2. 表征与编码
- 正弦位置编码
- 相对位置编码
- 可学习位置编码
- 空间/协方差/趋势特征提取

3. 训练与损失
- MSE、MAE、Gaussian NLL
- 图平滑损失（梯度约束）
- 物理约束损失
- 超参搜索：网格、随机、类贝叶斯、Optuna（可选）

4. 评估与优化
- RMSE、MAE、R²、MAPE、CRPS
- 基线对比、消融贡献、敏感性分析
- 批处理优化、内存优化、剪枝、量化、蒸馏

5. 系统集成
- `SpatialInterpolationIntegrator` 与 `realtime_interpolation` 融合
- 缓存与事件回调支持
- API：`/api/dl/spatial/train`、`/api/dl/spatial/predict`

## 使用入口

1. 训练脚本
```bash
python3 deep_learning/training/train_spatial_interpolation.py --model gnn --epochs 30
```

2. 推理脚本
```bash
python3 deep_learning/inference/run_spatial_interpolation_inference.py --model attention --grid-size 20
```

3. 示例
```bash
python3 deep_learning/examples/spatial_interpolation_demo.py
```
