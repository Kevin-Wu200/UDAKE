# 融合模型选择指南

## 概述

模型选择是融合模型系统的关键环节，决定了融合的最终性能和效率。本指南提供了系统的模型选择方法，帮助您从候选模型中选择最优组合，实现最佳融合效果。

## 选择原则

### 1. 多样性原则

**目标**：选择差异化的模型，避免冗余

**方法**：
- 不同架构（CNN、RNN、Transformer）
- 不同特征表示
- 不同训练策略
- 不同超参数配置

**评估指标**：
- 模型间相关性（理想值：0.5-0.8）
- 特征空间覆盖度
- 预测分布差异

**示例**：
```python
import numpy as np
from sklearn.metrics import pairwise_distances

def calculate_diversity(models):
    """计算模型多样性"""
    pred_matrix = np.array([model.predictions for model in models])

    # 1. 相关性多样性
    corr_matrix = np.corrcoef(pred_matrix)
    avg_corr = np.mean(corr_matrix[np.triu_indices_from(corr_matrix, k=1)])
    diversity_score = 1 - avg_corr

    # 2. 预测分布多样性
    distribution_diversity = np.std(pred_matrix, axis=0).mean()

    # 3. 特征空间多样性（如果有特征表示）
    if hasattr(models[0], 'features'):
        feature_matrix = np.array([model.features for model in models])
        feature_diversity = pairwise_distances(feature_matrix).mean()
    else:
        feature_diversity = 0

    return {
        'correlation_diversity': diversity_score,
        'distribution_diversity': distribution_diversity,
        'feature_diversity': feature_diversity,
        'overall_diversity': (diversity_score + distribution_diversity) / 2
    }

diversity = calculate_diversity(models)
print(f"模型多样性评分: {diversity['overall_diversity']:.4f}")
```

### 2. 性能原则

**目标**：选择高性能模型，确保融合基线

**方法**：
- 评估单模型性能（RMSE、MAE、R²）
- 分析模型稳定性
- 考虑计算效率

**评估指标**：
- 预测准确度（RMSE、MAE）
- 拟合优度（R²）
- 预测稳定性
- 计算效率

**示例**：
```python
def evaluate_model_performance(model, true_values):
    """评估单个模型性能"""
    pred = np.array(model.predictions)
    true = np.array(true_values)

    # 基础指标
    rmse = np.sqrt(np.mean((pred - true) ** 2))
    mae = np.mean(np.abs(pred - true))
    r2 = 1 - np.sum((true - pred) ** 2) / np.sum((true - true.mean()) ** 2)

    # 稳定性指标
    residuals = pred - true
    stability = 1 / (1 + np.std(residuals))

    # 效率指标（如果有运行时间）
    efficiency = 1.0  # 默认值

    return {
        'rmse': rmse,
        'mae': mae,
        'r2': r2,
        'stability': stability,
        'efficiency': efficiency
    }

# 评估所有模型
performances = []
for model in models:
    perf = evaluate_model_performance(model, true_values)
    performances.append(perf)
    print(f"{model.model_id}: RMSE={perf['rmse']:.4f}, R²={perf['r2']:.4f}")
```

### 3. 互补性原则

**目标**：选择互补的模型，覆盖不同场景

**方法**：
- 分析模型在不同区域的表现
- 识别模型的强项和弱项
- 评估错误模式的差异性

**评估指标**：
- 区域性能差异
- 错误模式相关性
- 互补性指数

**示例**：
```python
def analyze_complementarity(models, true_values, regions=None):
    """分析模型互补性"""
    if regions is None:
        # 如果没有区域信息，基于预测值划分
        all_preds = np.concatenate([model.predictions for model in models])
        percentiles = np.percentile(all_preds, [25, 50, 75])
        regions = np.digitize(all_preds, percentiles)

    n_regions = len(set(regions))
    n_models = len(models)

    # 计算每个模型在每个区域的性能
    regional_perf = np.zeros((n_models, n_regions))
    for i, model in enumerate(models):
        pred = np.array(model.predictions)
        true = np.array(true_values)
        for r in range(n_regions):
            mask = regions == r
            if mask.sum() > 0:
                regional_perf[i, r] = np.sqrt(np.mean((pred[mask] - true[mask]) ** 2))

    # 计算互补性：每个区域的最优模型
    best_models = np.argmin(regional_perf, axis=0)
    complementarity = len(set(best_models)) / n_models

    return {
        'regional_performance': regional_perf,
        'best_models_per_region': best_models,
        'complementarity_score': complementarity
    }

complementarity = analyze_complementarity(models, true_values)
print(f"互补性评分: {complementarity['complementarity_score']:.4f}")
```

### 4. 效率原则

**目标**：在保证性能的前提下，控制计算开销

**方法**：
- 评估模型推理时间
- 考虑模型大小和内存占用
- 平衡性能和效率

**评估指标**：
- 推理时间
- 内存占用
- 模型大小
- 资源利用率

**示例**：
```python
import time

def measure_model_efficiency(model, test_inputs, n_runs=10):
    """测量模型效率"""
    # 推理时间
    times = []
    for _ in range(n_runs):
        start_time = time.perf_counter()
        predictions = model.predict(test_inputs)  # 假设有predict方法
        end_time = time.perf_counter()
        times.append(end_time - start_time)

    avg_time = np.mean(times)

    # 模型大小（如果有）
    model_size = getattr(model, 'size', 0)

    # 内存占用（如果有）
    memory_usage = getattr(model, 'memory', 0)

    return {
        'inference_time': avg_time,
        'model_size': model_size,
        'memory_usage': memory_usage,
        'efficiency_score': 1.0 / (1.0 + avg_time)  # 简化计算
    }
```

## 选择方法

### 1. 贪心选择法

**思想**：逐步选择最优模型，每次选择对融合效果提升最大的模型

**算法**：
1. 初始化：选择性能最好的模型作为初始集合
2. 迭代：在剩余模型中，选择使融合性能提升最大的模型
3. 终止：当性能提升小于阈值或达到最大模型数时停止

**实现**：
```python
def greedy_model_selection(models, true_values, max_models=10, min_improvement=0.01):
    """贪心模型选择"""
    selected = []
    remaining = models.copy()
    best_rmse = float('inf')

    while remaining and len(selected) < max_models:
        best_candidate = None
        best_improvement = 0

        for candidate in remaining:
            temp_selected = selected + [candidate]

            # 计算融合性能
            config = FusionConfig(strategy=FusionStrategy.WEIGHTED_AVERAGE)
            result = engine.fuse(temp_selected, config=config, true_values=true_values)
            current_rmse = result.metrics['rmse']

            improvement = (best_rmse - current_rmse) / best_rmse if best_rmse != float('inf') else 0

            if improvement > best_improvement:
                best_improvement = improvement
                best_candidate = candidate

        if best_improvement < min_improvement:
            print(f"提升不足（{best_improvement:.2%}），停止选择")
            break

        selected.append(best_candidate)
        remaining.remove(best_candidate)

        # 更新最佳性能
        config = FusionConfig(strategy=FusionStrategy.WEIGHTED_AVERAGE)
        result = engine.fuse(selected, config=config, true_values=true_values)
        best_rmse = result.metrics['rmse']

        print(f"选择模型: {best_candidate.model_id}, "
              f"当前模型数: {len(selected)}, "
              f"RMSE: {best_rmse:.4f}, "
              f"提升: {best_improvement:.2%}")

    return selected, best_rmse

# 使用示例
selected_models, final_rmse = greedy_model_selection(models, true_values)
print(f"\n最终选择了 {len(selected_models)} 个模型")
print(f"最终RMSE: {final_rmse:.4f}")
```

### 2. 基于排名的选择法

**思想**：根据综合评分排名，选择前N个模型

**评分函数**：
```python
def rank_models(models, true_values, weights=None):
    """对模型进行排名"""
    if weights is None:
        weights = {
            'performance': 0.4,
            'diversity': 0.3,
            'stability': 0.2,
            'efficiency': 0.1
        }

    # 评估所有指标
    evaluations = []
    for model in models:
        perf = evaluate_model_performance(model, true_values)
        evaluations.append({
            'model': model,
            'performance': -perf['rmse'],  # 负RMSE，越大越好
            'stability': perf['stability'],
            'efficiency': perf['efficiency']
        })

    # 计算多样性（需要其他模型作为参考）
    for i, eval_i in enumerate(evaluations):
        diversities = []
        for j, eval_j in enumerate(evaluations):
            if i != j:
                corr = np.corrcoef(
                    eval_i['model'].predictions,
                    eval_j['model'].predictions
                )[0, 1]
                diversities.append(1 - corr)
        eval_i['diversity'] = np.mean(diversities)

    # 归一化指标
    for key in ['performance', 'diversity', 'stability', 'efficiency']:
        values = [e[key] for e in evaluations]
        min_val, max_val = min(values), max(values)
        for eval in evaluations:
            if max_val > min_val:
                eval[f'{key}_normalized'] = (eval[key] - min_val) / (max_val - min_val)
            else:
                eval[f'{key}_normalized'] = 0.5

    # 计算综合评分
    for eval in evaluations:
        eval['overall_score'] = (
            weights['performance'] * eval['performance_normalized'] +
            weights['diversity'] * eval['diversity_normalized'] +
            weights['stability'] * eval['stability_normalized'] +
            weights['efficiency'] * eval['efficiency_normalized']
        )

    # 排序
    evaluations.sort(key=lambda x: x['overall_score'], reverse=True)

    return evaluations

# 使用示例
ranked_models = rank_models(models, true_values)
print("模型排名:")
for i, eval in enumerate(ranked_models, 1):
    print(f"{i}. {eval['model'].model_id}: 综合评分={eval['overall_score']:.4f}")
```

### 3. 基于聚类的选择法

**思想**：将相似模型聚类，从每个聚类中选择代表模型

**算法**：
```python
from sklearn.cluster import KMeans

def cluster_based_selection(models, n_clusters=3, true_values=None):
    """基于聚类的模型选择"""
    # 构建特征矩阵
    pred_matrix = np.array([model.predictions for model in models])

    # 标准化
    pred_matrix = (pred_matrix - pred_matrix.mean()) / pred_matrix.std()

    # 聚类
    kmeans = KMeans(n_clusters=n_clusters, random_state=42)
    labels = kmeans.fit_predict(pred_matrix)

    # 从每个聚类中选择最优模型
    selected = []
    for cluster_id in range(n_clusters):
        cluster_models = [models[i] for i in range(len(models)) if labels[i] == cluster_id]

        if true_values is not None:
            # 选择性能最好的
            best_model = min(cluster_models,
                           key=lambda m: evaluate_model_performance(m, true_values)['rmse'])
        else:
            # 选择最接近聚类中心的
            cluster_center = kmeans.cluster_centers_[cluster_id]
            distances = [np.linalg.norm(pred_matrix[i] - cluster_center)
                        for i in range(len(models)) if labels[i] == cluster_id]
            best_idx = np.argmin(distances)
            best_model = cluster_models[best_idx]

        selected.append(best_model)
        print(f"聚类 {cluster_id}: 选择模型 {best_model.model_id}")

    return selected, labels

# 使用示例
selected_models, labels = cluster_based_selection(models, n_clusters=3, true_values=true_values)
print(f"选择了 {len(selected_models)} 个代表性模型")
```

### 4. 基于遗传算法的选择法

**思想**：使用遗传算法搜索最优模型组合

**算法**：
```python
import random

def genetic_algorithm_selection(models, true_values, population_size=20,
                                generations=50, mutation_rate=0.1):
    """基于遗传算法的模型选择"""
    n_models = len(models)

    def create_individual():
        """创建个体（模型组合）"""
        n_selected = random.randint(2, min(10, n_models))
        return random.sample(range(n_models), n_selected)

    def fitness(individual):
        """计算适应度（融合性能）"""
        selected_models = [models[i] for i in individual]
        config = FusionConfig(strategy=FusionStrategy.WEIGHTED_AVERAGE)
        result = engine.fuse(selected_models, config=config, true_values=true_values)
        return -result.metrics['rmse']  # 负RMSE，越大越好

    def crossover(parent1, parent2):
        """交叉操作"""
        # 合并并去重
        child_genes = list(set(parent1 + parent2))
        # 限制数量
        if len(child_genes) > 10:
            child_genes = random.sample(child_genes, 10)
        return child_genes

    def mutate(individual):
        """变异操作"""
        if random.random() < mutation_rate:
            if len(individual) > 2 and random.random() < 0.5:
                # 删除一个模型
                individual.remove(random.choice(individual))
            elif len(individual) < n_models:
                # 添加一个模型
                available = [i for i in range(n_models) if i not in individual]
                if available:
                    individual.append(random.choice(available))
        return individual

    # 初始化种群
    population = [create_individual() for _ in range(population_size)]

    # 进化
    for generation in range(generations):
        # 评估适应度
        fitness_scores = [fitness(ind) for ind in population]

        # 选择（锦标赛选择）
        new_population = []
        for _ in range(population_size):
            candidates = random.sample(list(zip(population, fitness_scores)), 3)
            parent = max(candidates, key=lambda x: x[1])[0]
            new_population.append(parent.copy())

        # 交叉
        for i in range(0, population_size, 2):
            if i + 1 < population_size:
                new_population[i], new_population[i+1] = (
                    crossover(new_population[i], new_population[i+1]),
                    crossover(new_population[i+1], new_population[i])
                )

        # 变异
        population = [mutate(ind) for ind in new_population]

        # 记录最佳个体
        best_idx = np.argmax(fitness_scores)
        best_individual = population[best_idx]
        best_fitness = fitness_scores[best_idx]

        if generation % 10 == 0:
            print(f"代数 {generation}, 最佳适应度: {-best_fitness:.4f}")

    # 返回最优解
    selected_models = [models[i] for i in best_individual]
    print(f"\n最优组合包含 {len(selected_models)} 个模型")

    return selected_models

# 使用示例
selected_models = genetic_algorithm_selection(models, true_values)
```

## 评估与验证

### 1. 交叉验证

**目的**：验证选择的模型组合的泛化性能

**实现**：
```python
from sklearn.model_selection import KFold

def cross_validate_selection(models, data, n_splits=5):
    """交叉验证模型选择"""
    kf = KFold(n_splits=n_splits, shuffle=True, random_state=42)

    cv_results = []

    for fold, (train_idx, val_idx) in enumerate(kf.split(data)):
        print(f"Fold {fold + 1}/{n_splits}")

        # 划分数据
        train_data = data[train_idx]
        val_data = data[val_idx]

        # 在训练集上选择模型
        selected_models, _ = greedy_model_selection(models, train_data['true_values'])

        # 在验证集上评估
        config = FusionConfig(strategy=FusionStrategy.WEIGHTED_AVERAGE)
        result = engine.fuse(
            [model.predict(val_data['inputs']) for model in selected_models],
            config=config,
            true_values=val_data['true_values']
        )

        cv_results.append({
            'fold': fold,
            'n_models': len(selected_models),
            'rmse': result.metrics['rmse'],
            'mae': result.metrics['mae'],
            'r2': result.metrics['r2']
        })

    # 汇总结果
    avg_rmse = np.mean([r['rmse'] for r in cv_results])
    std_rmse = np.std([r['rmse'] for r in cv_results])

    print(f"\n交叉验证结果:")
    print(f"  平均RMSE: {avg_rmse:.4f} ± {std_rmse:.4f}")
    print(f"  平均模型数: {np.mean([r['n_models'] for r in cv_results]):.1f}")

    return cv_results
```

### 2. 留一法验证

**目的**：在数据较少时进行可靠验证

**实现**：
```python
def leave_one_out_validation(models, data):
    """留一法验证"""
    n_samples = len(data)
    results = []

    for i in range(n_samples):
        # 留一
        train_data = data[np.arange(n_samples) != i]
        val_data = data[i:i+1]

        # 选择模型
        selected_models, _ = greedy_model_selection(models, train_data['true_values'])

        # 评估
        config = FusionConfig(strategy=FusionStrategy.WEIGHTED_AVERAGE)
        result = engine.fuse(
            [model.predict(val_data['inputs']) for model in selected_models],
            config=config,
            true_values=val_data['true_values']
        )

        results.append(result.metrics['rmse'])

        if (i + 1) % 10 == 0:
            print(f"已验证 {i + 1}/{n_samples} 样本")

    avg_rmse = np.mean(results)
    print(f"\n留一法验证结果: 平均RMSE = {avg_rmse:.4f}")

    return avg_rmse
```

### 3. 增量验证

**目的**：验证模型选择对新增数据的适应性

**实现**：
```python
def incremental_validation(models, initial_data, new_data_batches):
    """增量验证"""
    current_data = initial_data.copy()
    selected_models = []

    results = []

    for batch_idx, new_data in enumerate(new_data_batches):
        print(f"批次 {batch_idx + 1}")

        # 更新数据
        current_data = np.concatenate([current_data, new_data])

        # 重新选择模型
        selected_models, rmse = greedy_model_selection(
            models,
            current_data['true_values']
        )

        results.append({
            'batch': batch_idx,
            'n_models': len(selected_models),
            'rmse': rmse,
            'selected_models': [m.model_id for m in selected_models]
        })

    # 分析结果
    print("\n增量验证结果:")
    for r in results:
        print(f"批次 {r['batch']}: 模型数={r['n_models']}, RMSE={r['rmse']:.4f}")

    return results
```

### 1. 模型选择报告生成器

```python
def generate_selection_report(models, selected_models, true_values):
    """生成模型选择报告"""
    report = {
        'timestamp': time.time(),
        'total_candidates': len(models),
        'selected_count': len(selected_models),
        'selection_ratio': len(selected_models) / len(models)
    }

    # 评估所有模型
    all_evaluations = []
    for model in models:
        eval_result = evaluate_model_performance(model, true_values)
        all_evaluations.append({
            'model_id': model.model_id,
            'rmse': eval_result['rmse'],
            'mae': eval_result['mae'],
            'r2': eval_result['r2'],
            'stability': eval_result['stability'],
            'selected': model in selected_models
        })

    report['evaluations'] = all_evaluations

    # 融合性能
    config = FusionConfig(strategy=FusionStrategy.WEIGHTED_AVERAGE)
    fusion_result = engine.fuse(selected_models, config=config, true_values=true_values)

    report['fusion_performance'] = {
        'rmse': fusion_result.metrics['rmse'],
        'mae': fusion_result.metrics['mae'],
        'r2': fusion_result.metrics['r2'],
        'weights': fusion_result.weights
    }

    # 性能对比
    best_single = min(all_evaluations, key=lambda x: x['rmse'])
    report['performance_comparison'] = {
        'best_single_rmse': best_single['rmse'],
        'fusion_rmse': fusion_result.metrics['rmse'],
        'improvement': (best_single['rmse'] - fusion_result.metrics['rmse']) / best_single['rmse'] * 100
    }

    return report

# 使用示例
report = generate_selection_report(models, selected_models, true_values)

        axes[1, 1].hist(model.predictions, alpha=0.5, label=model.model_id, bins=20)
    axes[1, 1].set_xlabel('预测值')
    axes[1, 1].set_ylabel('频数')
    axes[1, 1].set_title('预测分布')
    axes[1, 1].legend()

    plt.tight_layout()
    plt.savefig('model_selection_visualization.png', dpi=300, bbox_inches='tight')
    plt.show()
```

## 最佳实践

### 1. 选择流程

```
1. 数据准备
   ├─ 数据清洗和标准化
   ├─ 划分训练/验证/测试集
   └─ 准备候选模型

2. 初步筛选
   ├─ 评估单模型性能
   ├─ 检查模型质量
   └─ 剔除无效模型

3. 多样性分析
   ├─ 计算模型相关性
   ├─ 分析特征覆盖
   └─ 识别冗余模型

4. 模型选择
   ├─ 应用选择算法
   ├─ 考虑多个因素
   └─ 选择最优组合

5. 验证评估
   ├─ 交叉验证
   ├─ 性能分析
   └─ 稳定性测试

6. 优化调整
   ├─ 参数调优
   ├─ 模型微调
   └─ 迭代改进
```

### 2. 选择建议

**小规模数据集（< 1000样本）**：
- 使用留一法验证
- 选择2-3个模型
- 优先考虑简单策略

**中等规模数据集（1000-10000样本）**：
- 使用交叉验证
- 选择3-5个模型
- 可以尝试复杂策略

**大规模数据集（> 10000样本）**：
- 使用抽样验证
- 选择5-10个模型
- 充分利用复杂策略

### 3. 常见陷阱

**避免过度选择**：
- 不要选择过多模型（> 10）
- 考虑边际收益递减
- 平衡性能和效率

**避免单一标准**：
- 不要只看RMSE
- 综合考虑多样性、稳定性
- 评估计算开销

**避免静态选择**：
- 定期重新评估
- 适应数据分布变化
- 持续优化组合
