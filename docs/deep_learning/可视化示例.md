# 可视化示例

## 预测对比

- 输入：采样点 `samples=[[x, y, value], ...]`
- 输出：
  - `prediction`：预测值序列
  - `variance`：不确定性序列

## 误差热力图

1. 使用 `predict_grid` 生成网格预测。
2. 与验证网格真值做差，得到绝对误差矩阵。
3. 使用前端热力图组件或 `matplotlib.imshow` 展示。

## 注意力权重

`AttentionKrigingModel.forward` 输出 `attention_weights`，可用于：
- 查询点重要邻域分析
- 模型可信度解释

## 不确定性可视化

- 将 `variance` 映射为颜色深浅
- 与预测均值联合展示风险区域
