# 时空预测模块（阶段6）

## 模块组成

- 模型：`SpatioTemporalTransformer`、`GCNLSTMModel`、`ConvLSTMModel`、`STGCNModel`
- 数据：时空数据集生成、增强、特征工程、标准化、动态图构建、滑动窗口加载
- 训练：余弦退火 + 预热、早停、梯度裁剪代理、模型保存/加载
- 推理：批量预测、在线滑窗预测、增量更新/微调
- 评估：RMSE/MAE/MAPE/R²/sMAPE/CRPS、时间维/空间维评估、基准对比、消融报告
- 分析：趋势/季节性分解、ADF/KPSS代理检验、ACF/PACF、FFT、时空异常检测

## CLI

```bash
python3 deep_learning/training/train_spatiotemporal.py --model st_transformer --epochs 20 --seq-len 24 --horizon 6
python3 deep_learning/inference/run_spatiotemporal_inference.py --model stgcn --seq-len 24 --horizon 6
```

## 后端 API

- `POST /api/dl/spatiotemporal/train`
- `POST /api/dl/spatiotemporal/predict`
- `POST /api/dl/spatiotemporal/online-update`

### 训练请求示例

```json
{
  "model_type": "st_transformer",
  "coords": [[0.1, 0.2], [0.4, 0.7]],
  "series": [
    [[1.0, 0.2], [1.1, 0.1]],
    [[0.9, 0.0], [1.0, -0.1]]
  ],
  "targets": [[1.2, 1.25], [1.05, 1.1]],
  "epochs": 20,
  "pred_horizon": 2
}
```
