## 异常检测模型概述

### 什么是异常检测？

异常检测是识别与大多数数据显著不同的数据点的过程，也称为离群点检测。在空间数据分析中，异常检测可以帮助识别：

- **数据质量问题**：测量错误、传感器故障
- **特殊现象**：污染热点、地质异常
- **模型偏差**：插值预测不准确的区域

### UDAKE异常检测模型体系

UDAKE平台提供四种深度学习异常检测模型，每种模型都有其独特的优势和适用场景：

#### 1. VAE异常检测（变分自编码器）

**核心原理**：
VAE通过学习数据的潜在分布，能够生成新的数据样本。异常数据通常在潜在空间中分布异常，导致重建误差较大。

**优势**：
- 基于概率框架，提供不确定性估计
- 潜在空间可视化，便于理解
- 对高维数据处理效果好

**适用场景**：
- 数据分布相对平滑
- 需要理解异常的原因
- 需要概率解释

**代码位置**：`/deep_learning/models/anomaly_detection/vae_anomaly.py`

#### 2. GCAE异常检测（图卷积自编码器）

**核心原理**：
GCAE考虑空间数据的图结构特性，通过图卷积捕捉空间依赖关系，同时检测节点、边和子图级别的异常。

**优势**：
- 充分利用空间结构信息
- 支持多粒度异常检测（节点/边/子图）
- 对空间异常敏感

**适用场景**：
- 空间数据具有明显的空间相关性
- 需要识别空间聚集异常
- 需要理解异常的空间影响

**代码位置**：`/deep_learning/models/anomaly_detection/gcae_anomaly.py`

#### 3. GAN异常检测（生成对抗网络）

**核心原理**：
GAN通过生成器和判别器的对抗训练，学习正常数据的分布。异常数据难以被生成器重建，且判别器评分异常。

**优势**：
- 生成能力强，对复杂数据分布效果好
- 判别器提供额外的异常信号
- 空间梯度分析

**适用场景**：
- 数据分布复杂
- 需要检测分布外样本
- 需要模式崩溃检测

**代码位置**：`/deep_learning/models/anomaly_detection/gan_anomaly.py`

#### 4. 对比学习异常检测

**核心原理**：
对比学习通过学习数据的特征表示，将正常样本聚集在特征空间中，异常样本远离正常样本。

**优势**：
- 无监督学习，无需标注数据
- 特征表示学习能力强
- 密度估计和最近邻分析

**适用场景**：
- 缺乏标注数据
- 需要在线异常检测
- 需要特征银行相似度

**代码位置**：`/deep_learning/models/anomaly_detection/contrastive_anomaly.py`

## LIME解释方法说明

### 什么是LIME？

LIME（Local Interpretable Model-agnostic Explanations）是一种局部可解释模型无关的解释方法，通过在局部构建线性模型来解释复杂模型的预测。

### LIME在异常检测中的应用

#### 1. 解释异常检测决策

LIME可以解释为什么某个数据点被判定为异常：

```python
from lime.lime_tabular import LimeTabularExplainer

# 创建LIME解释器
explainer = LimeTabularExplainer(
    training_data=normal_samples,
    feature_names=['x', 'y', 'value'],
    mode='regression'
)

# 解释异常检测决策
explanation = explainer.explain_instance(
    data_point=anomaly_sample,
    predict_fn=detector.anomaly_scores,
    num_features=5
)

# 显示解释
explanation.show_in_notebook()
```

#### 2. 识别关键特征

LIME可以识别哪些特征对异常检测决策贡献最大：

```python
# 获取特征重要性
feature_importance = explanation.as_list()

# 输出特征重要性
for feature, importance in feature_importance:
    print(f"{feature}: {importance:.4f}")
```

#### 3. 局部近似

LIME在异常点附近构建局部线性模型，解释异常检测模型的局部行为：

```python
# 局部近似解释
local_approx = explanation.local_pred

# 与全局模型对比
global_pred = detector.predict(data_point)
print(f"局部近似: {local_approx:.4f}")
print(f"全局预测: {global_pred:.4f}")
```

### LIME解释的可视化

```python
import matplotlib.pyplot as plt

# 特征重要性可视化
features, importances = zip(*explanation.as_list())
plt.barh(features, importances)
plt.xlabel('Importance')
plt.title('Feature Importance for Anomaly Detection')
plt.show()
```

### LIME的优势和局限

**优势**：
- 模型无关，适用于任何异常检测模型
- 局部解释准确，易于理解
- 可视化直观

**局限**：
- 解释范围局限于局部
- 对高维数据处理效果一般
- 解释稳定性可能不一致

## SHAP解释方法说明

### 什么是SHAP？

SHAP（SHapley Additive exPlanations）基于博弈论中的Shapley值，提供一致且局部的特征重要性解释。

### SHAP在异常检测中的应用

#### 1. 计算Shapley值

```python
import shap

# 创建SHAP解释器
explainer = shap.KernelExplainer(detector.anomaly_scores, normal_samples)

# 计算Shapley值
shap_values = explainer.shap_values(anomaly_sample)
```

#### 2. 特征重要性分析

```python
# 特征重要性可视化
shap.summary_plot(shap_values, normal_samples, feature_names=['x', 'y', 'value'])
```

#### 3. 个体异常解释

```python
# 个体解释
shap.force_plot(
    explainer.expected_value,
    shap_values,
    anomaly_sample,
    feature_names=['x', 'y', 'value']
)
```

#### 4. 依赖关系分析

```python
# 特征依赖关系分析
shap.dependence_plot(
    'value',
    shap_values,
    normal_samples,
    feature_names=['x', 'y', 'value']
)
```

### SHAP的优势和局限

**优势**：
- 理论基础扎实，基于博弈论
- 提供一致的特征重要性
- 支持全局和局部解释

**局限**：
- 计算复杂度高
- 对大样本集处理较慢
- 需要较多计算资源

## 异常检测参数说明

### VAE参数配置

```python
@dataclass
class VAETrainConfig:
    latent_dim: int = 4          # 潜在空间维度
    beta: float = 0.1            # KL散度权重
    max_epochs: int = 40         # 最大训练轮数
    noise_schedule_start: float = 1.0
    noise_schedule_end: float = 0.05
    batch_size: int = 32
    learning_rate: float = 0.001
    early_stopping_patience: int = 10
```

**参数说明**：

- `latent_dim`: 潜在空间维度，影响模型表达能力
  - 较小值：模型欠拟合，异常检测不敏感
  - 较大值：模型过拟合，异常检测过于敏感
  - 推荐值：4-8

- `beta`: KL散度权重，控制潜在空间的正则化强度
  - 较小值：潜在空间更灵活，可能过拟合
  - 较大值：潜在空间更约束，可能欠拟合
  - 推荐值：0.05-0.2

- `max_epochs`: 最大训练轮数
  - 较小值：训练不充分
  - 较大值：可能过拟合
  - 推荐值：30-50

### GCAE参数配置

```python
@dataclass
class GCAEConfig:
    latent_dim: int = 4
    knn_k: int = 8               # K近邻参数
    structure_weight: float = 0.4  # 结构损失权重
    feature_weight: float = 0.6   # 特征损失权重
    hidden_dim: int = 16
    max_epochs: int = 40
```

**参数说明**：

- `knn_k`: K近邻参数，影响图结构
  - 较小值：图结构稀疏，局部特征强
  - 较大值：图结构密集，全局特征强
  - 推荐值：5-10

- `structure_weight`: 结构损失权重
  - 较小值：特征重建为主
  - 较大值：结构重建为主
  - 推荐值：0.3-0.5

- `feature_weight`: 特征损失权重
  - 推荐值：0.5-0.7

### GAN参数配置

```python
@dataclass
class GANConfig:
    latent_dim: int = 6
    max_epochs: int = 60
    gp_weight: float = 10.0      # 梯度惩罚权重
    discriminator_steps: int = 5
    generator_steps: int = 1
```

**参数说明**：

- `gp_weight`: 梯度惩罚权重，用于WGAN-GP
  - 较小值：训练不稳定
  - 较大值：训练过于约束
  - 推荐值：5-15

- `discriminator_steps`: 判别器训练步数
  - 推荐值：3-5

### 对比学习参数配置

```python
@dataclass
class ContrastiveConfig:
    feature_dim: int = 12
    projection_dim: int = 8
    temperature: float = 0.2     # 温度参数
    queue_size: int = 256        # 特征队列大小
    max_epochs: int = 50
```

**参数说明**：

- `temperature`: 温度参数，控制softmax的平滑度
  - 较小值：更尖锐的分布，更容易区分
  - 较大值：更平滑的分布，更难区分
  - 推荐值：0.1-0.5

- `queue_size`: 特征队列大小
  - 较小值：特征库不完整
  - 较大值：内存消耗大
  - 推荐值：128-512

## 解释结果解读指南

### 异常分数解读

#### 1. VAE异常分数

```python
# 异常分数计算
s(x) = 0.7 * ||x - x̂||² + 0.3 * ||z - μz||²
```

**解读**：
- `||x - x̂||²`: 重建误差，越大表示越异常
- `||z - μz||²`: 潜在空间距离，越大表示越异常
- 总分数越大，异常可能性越高

**阈值设置**：
- 使用正常样本的95分位数作为阈值
- 或使用自适应阈值：μ + 2σ

#### 2. GCAE异常分数

```python
# 异常分数计算
s(x) = α * s_node + β * s_edge + γ * s_subgraph
```

**解读**：
- `s_node`: 节点级异常分数
- `s_edge`: 边级异常分数
- `s_subgraph`: 子图级异常分数
- α, β, γ为权重，默认0.4, 0.3, 0.3

#### 3. GAN异常分数

```python
# 异常分数计算
s(x) = 0.5 * (1 - D(x)) + 0.35 * ||x - G(z*)||² + 0.15 * ||∇xD(x)||²
```

**解读**：
- `1 - D(x)`: 判别器分数，越大表示越异常
- `||x - G(z*)||²`: 重建误差
- `||∇xD(x)||²`: 空间梯度分数

#### 4. 对比学习异常分数

```python
# 异常分数计算
s(x) = 0.4 * d_feature + 0.3 * ρ(x) + 0.2 * d_nn + 0.1 * s_bank
```

**解读**：
- `d_feature`: 特征空间距离
- `ρ(x)`: 密度估计
- `d_nn`: 最近邻距离
- `s_bank`: 特征银行相似度

### 潜在空间可视化解读

#### VAE潜在空间

```python
# 潜在空间可视化
latent_vis = detector.latent_visualization(normal_samples, anomaly_samples)
```

**解读**：
- 正常样本聚集在潜在空间中心
- 异常样本偏离中心区域
- 距离中心越远，异常可能性越高

#### 注意力权重解读

```python
# GCAE注意力权重
attention_weights = detector.get_attention_weights(data)
```

**解读**：
- 高权重表示对异常检测决策贡献大
- 可以识别异常的空间影响范围
- 支持交互式可视化

### 解释可靠性评估

#### 1. 解释一致性

多次运行解释方法，检查结果一致性：

```python
importances_multiple = []
for _ in range(10):
    exp = explainer.explain_instance(anomaly_sample, ...)
    importances_multiple.append(explanation.as_list())

# 计算标准差
import std_dev = np.std(importances_multiple, axis=0)
```

#### 2. 解释覆盖率

检查解释是否覆盖了所有重要特征：

```python
coverage = len(feature_importance) / total_features
print(f"解释覆盖率: {coverage:.2%}")
```

#### 3. 解释稳定性

添加小扰动，检查解释是否稳定：

```python
exp_original = explainer.explain_instance(anomaly_sample, ...)
exp_perturbed = explainer.explain_instance(anomaly_sample + noise, ...)

stability = similarity(exp_original, exp_perturbed)
print(f"解释稳定性: {stability:.2f}")
```

## 使用示例

### 示例1：VAE异常检测

```python
import numpy as np
from deep_learning.models.anomaly_detection.vae_anomaly import VAEAnomalyDetector

# 1. 准备数据
# 正常样本：空间位置(x, y)和观测值
normal_samples = np.random.normal(0, 1, (1000, 3))

# 异常样本
anomaly_samples = np.random.normal(5, 1, (10, 3))

# 2. 配置参数
config = VAETrainConfig(
    latent_dim=4,
    beta=0.1,
    max_epochs=40,
    batch_size=32
)

# 3. 创建检测器
detector = VAEAnomalyDetector(config=config)

# 4. 训练
history = detector.fit(normal_samples)

# 5. 检测异常
anomaly_scores = detector.anomaly_scores(anomaly_samples)
predictions = detector.predict(anomaly_samples)

# 6. 解释异常
from lime.lime_tabular import LimeTabularExplainer
explainer = LimeTabularExplainer(
    normal_samples,
    feature_names=['x', 'y', 'value'],
    mode='regression'
)

for i, anomaly in enumerate(anomaly_samples):
    explanation = explainer.explain_instance(
        anomaly,
        detector.anomaly_scores,
        num_features=3
    )
    print(f"异常点 {i+1}:")
    print(explanation.as_list())

# 7. 可视化
detector.latent_visualization(normal_samples, anomaly_samples)
```

### 示例5：集成异常检测

```python
from deep_learning.models.anomaly_detection.vae_anomaly import VAEAnomalyDetector

normal_samples = np.random.normal(0, 1, (1000, 3))

ensemble_scores = []

ensemble_scores.append(gan_detector.anomaly_scores(anomaly_samples))
ensemble_scores.append(contrastive_detector.anomaly_scores(anomaly_samples))

# 5. 投票
final_predictions = np.mean(ensemble_scores, axis=0) > threshold

# 6. 一致性分析
consistency = np.std(ensemble_scores, axis=0)
print(f"预测一致性: {consistency}")
```

## 常见问题解答

### Q1: 如何选择合适的异常检测模型？

**A**: 根据数据特点和应用场景选择：

- **VAE**: 数据分布相对平滑，需要概率解释
- **GCAE**: 空间相关性强，需要识别空间异常
- **GAN**: 数据分布复杂，需要检测分布外样本
- **对比学习**: 缺乏标注数据，需要在线检测
- **集成方法**: 追求最高精度，可以接受较慢速度

### Q2: 异常检测模型的准确率如何评估？

**A**: 使用以下评估指标：

```python
from sklearn.metrics import accuracy_score, precision_score, recall_score, f1_score

# 假设有真实标签

y_pred = [0, 0, 1, 1, 1, 0]

# 计算指标
accuracy = accuracy_score(y_true, y_pred)
precision = precision_score(y_true, y_pred)
recall = recall_score(y_true, y_pred)
f1 = f1_score(y_true, y_pred)

print(f"准确率: {accuracy:.2%}")
print(f"精确率: {precision:.2%}")
print(f"召回率: {recall:.2%}")
print(f"F1分数: {f1:.2%}")
```

### Q3: 如何设置异常检测阈值？

**A**: 有几种常用方法：

```python
# 方法1: 基于百分位数
threshold = np.percentile(normal_scores, 95)

# 方法2: 基于均值和标准差
threshold = np.mean(normal_scores) + 2 * np.std(normal_scores)

# 方法3: 基于ROC曲线优化
from sklearn.metrics import roc_curve
fpr, tpr, thresholds = roc_curve(y_true, scores)
optimal_idx = np.argmax(tpr - fpr)
threshold = thresholds[optimal_idx]
```

### Q5: LIME和SHAP解释如何选择？

**A**: 根据需求选择：

- **LIME**: 需要快速局部解释，模型复杂度高
- **SHAP**: 需要一致的全局解释，理论基础扎实
