# 强化学习采样优化模块

## 概述

`deep_learning/models/sampling_rl` 提供阶段5强化学习采样优化能力，覆盖：

- 环境：`SamplingEnv`（观测空间、离散/连续/混合动作、奖励、状态转移、终止、重置、渲染）
- 智能体：`PPOAgent`、`DQNAgent`、`ActorCriticAgent(A2C/A3C)`
- 多智能体：`MultiAgentSamplingSystem`（协作/竞争、QMIX、MADDPG）
- 奖励与特征：`rewards.py`、`features.py`
- 训练与评估：`training.py`、`evaluation.py`
- 在线学习：`online.py`（增量更新、持续学习、迁移与元学习）
- 性能优化：`optimization.py`（批处理、内存、GPU、推理缓存、并行采样）
- 集成：`integration.py`（与现有 adaptive sampling 流程融合）
