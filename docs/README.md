# UDAKE 项目文档

## 项目简介

UDAKE（Uncertainty-Driven Adaptive Kriging Engine）是一个智能不确定性驱动的自适应克里金引擎平台。

项目核心目标围绕地统计建模过程中的不确定性量化、模型自适应选择与采样策略优化进行系统级创新。通过自动变异函数拟合、LOOCV 误差评估、不确定性场分析与动态采样推荐机制，实现模型—误差—采样的闭环迭代更新。

### 核心创新点

- 不确定性驱动的自适应采样策略
- 基于交叉验证的模型自动选择机制
- 实时误差热区识别与收敛分析
- 面向工程应用的可解释决策流程

## 技术栈

### 后端

- FastAPI：高性能异步 Web 框架
- PyKrige：克里金插值库
- GSTools：地统计工具
- GDAL：地理空间数据处理

### 前端

- 原生 JavaScript（ES6 模块）
- 地图引擎：高德地图 / GeoScene / 天地图（可切换）
- CSS3：Apple 风格设计系统
- Electron：桌面应用框架
