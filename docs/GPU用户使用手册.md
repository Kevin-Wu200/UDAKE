# GPU用户使用手册

## 1. 功能说明
GPU加速系统支持矩阵运算、向量运算和克里金关键计算加速，并提供任务状态与性能指标查询。

## 2. 使用流程
1. 访问 `GET /api/gpu/health` 检查可用性。
2. 通过 `PUT /api/gpu/config` 设置 GPU 策略。
3. 调用计算接口执行任务。
4. 通过 `GET /api/gpu/tasks` 和 `GET /api/gpu/metrics` 查看运行状态与性能。

## 3. 常见问题
- GPU 不可用：系统会自动回退到 CPU。
- 任务失败：查看响应中的错误信息并检查输入维度。
- 性能不达预期：增大任务规模阈值或开启自动切换策略。
