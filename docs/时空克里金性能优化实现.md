# 时空克里金性能优化实现说明

更新时间：2026-04-05

## 已实现优化项

### 1. 分块与窗口
- 空间四叉树递归分块：`SpatiotemporalKrigingSolver.spatial_blocks`
- 边界重叠处理：`overlap_ratio`
- 时间滑动窗口：`temporal_windows`
- 窗口结果合并：`merge_window_predictions`

### 2. 低秩近似与秩优化
- 地标点采样：`sample_landmarks`
- Nystrom近似：`nystrom_approximation`
- Woodbury逆矩阵：`woodbury_inverse`
- 自适应秩选择：`adaptive_rank`

### 3. 矩阵计算优化
- Cholesky求解（jitter回退）：`solve_cholesky`
- 稀疏求解（SciPy可选）：`solve_sparse_cholesky`
- 并行多右端求解：`solve_parallel_cholesky`
- GPU/CPU自动切换（CuPy可选）：`maybe_gpu_matrix_multiply`

### 4. 增量更新优化
- Sherman-Morrison：`sherman_morrison_update`
- Woodbury批量更新：`woodbury_batch_update`
- 增量参数漂移调整：`incremental_update`

### 5. 缓存优化
- L1/L2：复用统一缓存服务
- L3磁盘缓存：`DiskCacheStore`
- 缓存预热：`warm_cache`
- 预取候选：`prefetch_candidates`
- 智能缓存策略：`smart_cache_policy`

### 6. 内存与性能监控
- 性能指标记录：`SpatiotemporalPerformanceMonitor`
- 内存快照与预警：`SpatiotemporalMemoryManager`
- 内存增长率估计：`leak_growth_rate`
- API查询：`GET /api/spatiotemporal/performance/metrics`
