# 时空克里金技术架构与功能模块实现说明

更新时间：2026-04-05

## 1. 模块实现概览

本实现对应 `TODOList/时空克里金分析模块/02_技术架构与功能模块.txt` 的功能模块清单，落地以下后端核心类：

- `SpatiotemporalVariogramModeler`
- `SpatiotemporalKrigingSolver`
- `SpatiotemporalModelAutoSelector`
- `SpatiotemporalPredictionEngine`
- `IncrementalSTKrigingEngine`

代码位置：`services/backend/app/services/spatiotemporal_core.py`

聚合服务位置：`services/backend/app/services/spatiotemporal_kriging_service.py`

API 路由位置：`services/backend/app/api/时空克里金接口.py`

## 2. 变异函数建模器（4.1）

### 已实现能力

- 时空数据预处理：缺失值填充、归一化
- 空间变异函数估计：经验半变异统计
- 时间变异函数估计：按时间滞后统计
- 时空协方差建模：支持 `separated/product/nonseparable`
- 参数估计：基于近似 MLE 的网格搜索
- 输出拟合报告和可视化图表数据

### 输出字段

- `parameters`
- `fitting_report`
- `charts.spatial_variogram`
- `charts.temporal_variogram`

## 3. 时空克里金求解器（4.2）

### 已实现能力

- 协方差矩阵计算
- 空间分块（四叉树式递归切分 + 可选重叠）
- 时间窗口切分
- Nystrom 低秩近似
- Cholesky 求解（含 jitter 回退）
- 权重、预测值、方差计算

## 4. 模型自动选择器（4.3 + 5.2）

### 已实现指标

- `RMSE`
- `MAE`
- `CRPS`
- 不确定性校准分数（95% 覆盖率偏差）
- 综合评分（加权和，分数越小越优）

### 自动选择流程

1. 计算候选模型评估指标
2. 生成综合评分
3. 选择最小评分模型
4. 生成排名报告

## 5. 预测引擎（4.4）

### 已实现能力

- 实时预测（在线模式）
- 离线分析（后端不可用降级）
- 精度衰减曲线：
  - 1 天：5%
  - 7 天：15%
  - 15 天：30%
- 在线/离线模式切换
- 预测结果缓存

## 6. 增量更新引擎（4.5）

### 已实现能力

- 新样本增量合并
- Sherman-Morrison 公式接口
- 参数漂移自适应调整
- 更新报告输出

### API

- `POST /api/spatiotemporal/incremental-update`
