# 克里金算法与模型选择说明

## 克里金插值原理

克里金（Kriging）是一种基于地统计学的空间插值方法，通过分析空间自相关性来预测未知位置的值。

## 支持的克里金方法

### 1. 普通克里金（Ordinary Kriging）

**适用场景**: 数据无明显趋势，空间平稳

**特点**:
- 假设均值未知但恒定
- 最常用的克里金方法
- 计算效率高

**数学模型**:
```
Z(s) = μ + ε(s)
```

### 2. 泛克里金（Universal Kriging）

**适用场景**: 数据存在空间趋势

**特点**:
- 考虑空间趋势项
- 适合大范围数据
- 精度更高

**数学模型**:
```
Z(s) = m(s) + ε(s)
m(s) = Σ βk * fk(s)
```

### 3. 分块克里金（Block Kriging）

**适用场景**: 大数据集，需要快速计算

**特点**:
- 分块处理数据
- 提高计算效率
- 适合实时应用

## 变异函数模型

### 1. 球状模型（Spherical）

**特点**: 最常用，适合大多数情况

**公式**:
```
γ(h) = c0 + c * [1.5(h/a) - 0.5(h/a)³]  (h ≤ a)
γ(h) = c0 + c  (h > a)
```

### 2. 指数模型（Exponential）

**特点**: 渐近达到基台值

**公式**:
```
γ(h) = c0 + c * [1 - exp(-3h/a)]
```

### 3. 高斯模型（Gaussian）

**特点**: 平滑性最好

**公式**:
```
γ(h) = c0 + c * [1 - exp(-3(h/a)²)]
```

## 模型选择建议

| 数据特征 | 推荐方法 | 推荐模型 |
|---------|---------|---------|
| 无趋势，点数<500 | 普通克里金 | 球状模型 |
| 有趋势，点数<500 | 泛克里金 | 指数模型 |
| 点数>1000 | 分块克里金 | 球状模型 |
| 数据平滑 | 普通克里金 | 高斯模型 |

## 参数调优

### 网格分辨率
- 低分辨率(50-100): 快速预览
- 中分辨率(100-200): 常规分析
- 高分辨率(200-500): 精细分析

### 滞后数（nlags）
- 点数<100: nlags=6
- 点数100-500: nlags=12
- 点数>500: nlags=20

## 交叉验证

使用K折交叉验证评估模型性能：
- RMSE: 均方根误差
- MAE: 平均绝对误差
- R²: 决定系数

建议R² > 0.7为良好模型。
