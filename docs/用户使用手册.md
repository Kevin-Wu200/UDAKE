# UDAKE 用户使用手册

## 克里金参数可视化

### 1. 功能入口

在参数配置区域点击“参数可视化”，可看到以下三个子模块：
- 参数关系图（`ParameterRelationshipChart`）
- 参数影响预览（`ParameterImpactPreview`）
- 实时变异函数拟合（`VariogramChart`）

### 2. 参数关系图（ParameterRelationshipChart）完整说明

#### 2.1 关系类型

1. `nugget-sill（块金值-基台值）`
- 含义：检查参数合法性，核心约束为 `nugget ≤ sill`。
- 状态解释：
  - 有效：满足约束。
  - 无效：违反约束，需要立即调整。

2. `range-spatial（范围值-空间范围）`
- 含义：将 `range` 与当前采样空间尺度联动。
- 推荐区：`range` 处于空间对角线长度的 `15%~40%`。
- 状态解释：
  - 有效：位于推荐区。
  - 警告：超出推荐区，但不一定非法。

3. `grid-performance（网格分辨率-性能）`
- 含义：估计网格分辨率对计算耗时的影响。
- 推荐区：预估耗时 `< 10s`。
- 状态解释：
  - 有效：性能风险较低。
  - 警告：可能导致运行时延过高。

#### 2.2 交互能力

- 悬浮提示：鼠标移动到当前点/历史轨迹点时显示 `X`、`Y` 精确值和状态（有效/警告/无效）。
- 触摸提示：移动端触摸可触发同样提示。
- 点击回写：点击关系图任意位置可将坐标回写到对应参数。
- 历史轨迹：记录最近调整路径，便于回溯调参过程。
- 一键恢复：当状态为警告/无效时，可点“恢复到推荐区”自动回到推荐区中心点。

### 4. 变异函数拟合（VariogramChart）参数调整指南

#### 4.1 参数解释

- `nugget`：短尺度随机性/测量噪声；过大易造成局部跳变。
- `sill`：总体方差上限；决定半方差平台。
- `range`：空间相关影响距离；过小会导致图面碎片化，过大可能过度平滑。
- `modelType`：
  - `spherical`：默认稳健，适合多数场景。
  - `exponential`：短距变化快，适合局地突变数据。
  - `gaussian`：曲线更平滑，适合连续地形场。
  - `linear`：快速探索用，不建议作为最终结果。

#### 4.2 调参顺序建议

1. 固定模型类型，先把 `nugget` 调到不过高。
2. 调 `sill` 使拟合曲线平台接近经验点高位。
3. 调 `range` 让曲线拐点覆盖主要经验点分布区。
4. 查看拟合质量面板：优先关注 `R²` 与 `RMSE`。
5. 若 `R²` 低：先换模型，再微调 `range`。
6. 若 `RMSE` 高：优先减小 `nugget`，必要时增加 `nlags`。

### 5. 参数配置最佳实践示例（5组）

1. 小样本快速试算（< 30点）
- `grid-resolution=80, nlags=8, nugget=0.15, sill=1.0, range=20`
- `method=ordinary, variogram=spherical`

2. 常规平衡场景（30~150点）
- `grid-resolution=120, nlags=12, nugget=0.10, sill=1.0, range=30`
- `method=ordinary, variogram=spherical`

3. 连续地形精细建模（100~400点）
- `grid-resolution=180, nlags=16, nugget=0.08, sill=1.2, range=42`
- `method=universal, variogram=gaussian`

4. 大数据性能优先（> 400点）
- `grid-resolution=90, nlags=10, nugget=0.20, sill=1.2, range=35`
- `method=block, variogram=exponential`

5. 边界噪声明显场景
- `grid-resolution=140, nlags=14, nugget=0.25, sill=1.4, range=32`
- `method=ordinary, variogram=exponential`

### 6. FAQ（15问）

1. 问：关系图中的“警告”是不是错误？
答：不是。警告表示偏离推荐区，但仍可运行。

2. 问：什么时候会出现“无效”？
答：当前主要在 `nugget > sill` 时出现无效。

3. 问：为什么切换关系类型后历史轨迹清空？
答：坐标轴含义改变，保留旧轨迹会误导判断。

4. 问：移动端如何查看悬浮提示？
答：在图上触摸并移动即可显示。

5. 问：恢复推荐区后参数为何有动画过渡？
答：用于提供可感知的参数变化反馈，避免突变。

6. 问：`range-spatial` 的空间范围从哪里来？
答：由采样点包络框对角线自动计算。

7. 问：`grid-performance` 的耗时是实测吗？
答：是估算值，用于调参方向判断，不替代真实任务耗时。

8. 问：为什么导出的总是最新预览图？
答：导出按钮默认导出“最近一次生成”的结果。

9. 问：预览卡片最多保留多少条？
答：最多保留 6 条，超出会自动淘汰最早记录。

10. 问：拟合质量里的 `R²` 越大越好吗？
答：是，通常越接近 1 表示拟合越好。

11. 问：`RMSE` 多大算高？
答：与数据尺度相关；同一任务内对比时越小越好。

12. 问：什么时候应切换为 `gaussian` 模型？
答：当空间变化连续平滑，且 `spherical` 难以贴合时。

13. 问：模板导入失败常见原因是什么？
答：JSON 格式错误或字段缺失。

14. 问：为什么我保存的模板不在下次启动出现？
答：请检查浏览器本地存储是否被清理。

15. 问：如何在速度与精度之间折中？
答：优先使用“平衡模式”，然后按 `grid-resolution` 与 `range` 小步微调。
