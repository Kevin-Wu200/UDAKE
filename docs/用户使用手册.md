# UDAKE 用户使用手册

## 克里金参数可视化

### 1. 功能入口

在参数配置区域点击“参数可视化”，可看到：
- 参数关系图
- 参数影响预览
- 实时变异函数拟合

### 2. 参数关系图切换

在“关系类型”下拉框中选择：

1. `nugget-sill（块金值-基台值）`
- 目标：保证 `nugget ≤ sill`
- 适用：基础参数合法性校验

2. `range-spatial（范围值-空间范围）`
- 目标：`range` 保持在空间范围 15%-40%
- 适用：控制空间相关范围与平滑程度

3. `grid-performance（网格分辨率-性能）`
- 目标：预估耗时 < 10s
- 适用：性能优先场景

### 3. 操作步骤

1. 调整参数滑块或输入框。
2. 观察关系图中的当前位置和约束高亮。
3. 必要时直接点击关系图回写参数。
4. 点击“生成预览”比较质量/速度/内存指标。
5. 查看“实时变异函数拟合”建议后继续微调。

### 4. 常见问题

- 问：为什么关系图切换后轨迹重置？
- 答：不同关系图使用不同坐标轴，重置可避免历史轨迹误导。

- 问：`range-spatial` 的空间范围如何计算？
- 答：根据当前采样点包络框对角线长度自动计算。

- 问：`grid-performance` 的耗时是真实值吗？
- 答：是预估值，用于快速调参，最终以实际运行时间为准。

### 5. 参数配置示例

- 快速预览：`grid=100, nlags=10, nugget=0.1, sill=1.0, range=25`
- 平衡模式：`grid=120, nlags=12, nugget=0.1, sill=1.0, range=30`
- 高精度模式：`grid=220, nlags=18, nugget=0.05, sill=1.2, range=40`
