# 企业管理员使用指南

## 1. 角色说明
- 试用企业管理员：仅可创建 `enterprise_trial`。
- 标准企业管理员：可创建 `enterprise_trial` 与 `enterprise_standard`。

## 2. 密钥类型与配额
- 页面会展示已创建数量、剩余数量与配额进度条。
- 超过配额后创建按钮将被禁用并提示。

## 3. 密钥过期与续期
- 密钥列表提供过期时间和到期倒计时。
- 过期密钥会显示红色标识，并不可继续分配。
- 续期需联系平台超级管理员执行。

## 4. 邮件提醒
- 系统会在到期前按定时任务发送提醒邮件。
- 失败邮件可在“邮件发送记录”页面手动重发。
