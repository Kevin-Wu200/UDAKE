# 使用手册

## 概述

UDAKE 提供完整的空间插值与不确定性分析功能，支持数据上传、克里金插值、结果可视化和分析报告生成。

## 基本操作流程

1. 新建项目，选择采样模式（自由采样 / 区域采样）
2. 添加采样点或上传 GeoJSON 数据
3. 配置插值参数（克里金方法、变异函数模型、网格分辨率）
4. 启动插值计算
5. 查看预测结果和方差结果
6. 生成分析报告

## 克里金算法说明

克里金（Kriging）是一种基于地统计学的空间插值方法，通过分析空间自相关性来预测未知位置的值。

### 支持的克里金方法

**普通克里金（Ordinary Kriging）**

适用于数据无明显趋势、空间平稳的场景。假设均值未知但恒定，是最常用的克里金方法，计算效率高。

数学模型：`Z(s) = μ + ε(s)`

**泛克里金（Universal Kriging）**

适用于数据存在空间趋势的场景。考虑空间趋势项，适合大范围数据，精度更高。

数学模型：`Z(s) = m(s) + ε(s)`，其中 `m(s) = Σ βk * fk(s)`

**分块克里金（Block Kriging）**

适用于大数据集，需要快速计算的场景。分块处理数据，提高计算效率。

### 变异函数模型

**球状模型（Spherical）**

最常用，适合大多数情况。公式：
```
γ(h) = c0 + c * [1.5(h/a) - 0.5(h/a)³]  (h ≤ a)
γ(h) = c0 + c  (h > a)
```

**指数模型（Exponential）**

渐近达到基台值。公式：`γ(h) = c0 + c * [1 - exp(-3h/a)]`

**高斯模型（Gaussian）**

平滑性最好。公式：`γ(h) = c0 + c * [1 - exp(-3(h/a)²)]`

### 模型选择建议

| 数据特征 | 推荐方法 | 推荐模型 |
|---------|---------|---------|
| 无趋势，点数 < 500 | 普通克里金 | 球状模型 |
| 有趋势，点数 < 500 | 泛克里金 | 指数模型 |
| 点数 > 1000 | 分块克里金 | 球状模型 |
| 数据平滑 | 普通克里金 | 高斯模型 |

### 参数调优

**网格分辨率**
- 低分辨率（50-100）：快速预览
- 中分辨率（100-200）：常规分析
- 高分辨率（200-500）：精细分析

**滞后数（nlags）**
- 点数 < 100：nlags = 6
- 点数 100-500：nlags = 12
- 点数 > 500：nlags = 20

### 交叉验证

使用 K 折交叉验证评估模型性能：
- RMSE：均方根误差
- MAE：平均绝对误差
- R²：决定系数（建议 R² > 0.7 为良好模型）

## 不确定性量化

### 不确定性来源

- 测量不确定性：仪器误差、人为误差、环境干扰
- 空间不确定性：采样密度不足、空间异质性、边界效应
- 模型不确定性：变异函数拟合误差、插值方法假设、参数估计误差

### 不确定性指标

**克里金方差**

预测位置的估计方差：`σ²(s0) = c0 + c - Σλi * γ(si, s0) + μ`

值越大表示不确定性越高，用于置信区间计算。

**95% 置信区间**

`[Z(s0) - 1.96σ(s0), Z(s0) + 1.96σ(s0)]`

**风险指数**

`Risk = 0.7 * (σ²/σ²_max) + 0.3 * (|Z|/|Z|_max)`

分级：低风险（< 0.33）、中风险（0.33-0.67）、高风险（≥ 0.67）

### 不确定性分级

| 等级 | 归一化方差范围 | 建议 |
|-----|--------------|------|
| 很低 | [0, 0.2) | 可直接使用 |
| 低 | [0.2, 0.4) | 可使用 |
| 中等 | [0.4, 0.6) | 谨慎使用 |
| 高 | [0.6, 0.8) | 需补充采样 |
| 很高 | [0.8, 1.0] | 不建议使用 |

### 质量控制

- 交叉验证 RMSE 应小于预测值标准差的 30%
- 方差均值应小于预测值方差
- 方差空间分布应合理连续

## 地图引擎使用

系统支持高德地图、ArcGIS 和天地图三种地图引擎，可通过配置切换。

### 功能对比

| 功能 | 高德地图 | ArcGIS | 天地图 |
|------|---------|--------|--------|
| 地图加载 | 快速 | 中等 | 快速 |
| 采样点显示 | 支持 | 支持 | 支持 |
| GeoJSON 图层 | 支持 | 支持 | 支持 |
| 栅格图层 | 支持 | 原生支持 | 需插件 |
| 深色模式 | 不支持 | 自动切换 | 不支持 |
| 设备定位 | 支持 | 支持 | 不支持 |

### 常见问题

**地图无法加载**
1. 确认已选择正确的地图引擎
2. 检查浏览器控制台错误信息
3. 确认网络连接正常
4. 检查 Token / API Key 配置

**Token 错误**
- 未配置 Token 时会进入 Mock 模式，地图仍可使用
- 如需正式使用，请申请并配置真实 Token
