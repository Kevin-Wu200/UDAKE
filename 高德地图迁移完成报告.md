## 已知限制

### 1. 栅格图层支持
高德地图 JS API 不直接支持 GeoTIFF 栅格图层。`AMapAdapter.addRasterLayer()` 方法已预留，但功能未实现。

**解决方案**：
- 使用高德地图的自定义图层功能
- 或者将栅格数据转换为瓦片服务

### 2. RegionSampling 组件耦合
`frontend/js/sampling/RegionSampling.js` 直接使用了 ArcGIS API：
- 第 172-175 行：使用 `window.esri.require`
- 第 178-191 行：创建 ArcGIS Graphic
- 第 194-199 行：创建 ArcGIS GraphicsLayer

**影响**：
- 区域采样功能在高德地图模式下不可用
- 需要重构为适配器模式

**解决方案**：
- 将地图操作抽象到适配器层
- 或者为高德地图创建独立的 RegionSampling 组件

### 3. FreeSampling 组件
需要检查 `frontend/js/sampling/FreeSampling.js` 是否也有类似问题。
