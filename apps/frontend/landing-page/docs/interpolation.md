# 空间插值模块

## 模块简介

空间插值模块是UDAKE平台的核心功能之一，提供高精度的空间面构建能力。通过多种插值算法，能够将离散的采样点数据转换为连续的空间分布图，广泛应用于环境监测、地质勘探、农业规划等领域。

## 核心功能

### 1. 多种插值算法支持

- **克里金插值（Kriging）**
  - 普通克里金（Ordinary Kriging）
  - 通用克里金（Universal Kriging）
  - 块克里金（Block Kriging）

- **确定性插值方法**
  - 反距离加权（IDW）
  - 径向基函数（RBF）
  - 最近邻插值（Nearest Neighbor）

### 2. 变异函数建模

- **模型类型**
  - 球状模型（Spherical）
  - 指数模型（Exponential）
  - 高斯模型（Gaussian）
  - 线性模型（Linear）

- **参数自动估计**
  - 块金值（Nugget）
  - 基台值（Sill）
  - 变程（Range）

### 3. 参数可视化

- 参数关系图（nugget-sill、range-spatial、grid-performance）
- 参数影响预览
- 实时变异函数拟合
- 拟合质量评估（R²、RMSE）

### 4. 结果可视化

- 等值线图
- 热力图
- 3D地形图
- 不确定性可视化

## 配置说明

### 推荐参数配置

#### 小样本快速试算（< 30点）
```
grid-resolution: 80
nlags: 8
nugget: 0.15
sill: 1.0
range: 20
method: ordinary
variogram: spherical
```

#### 常规平衡场景（30~150点）
```
grid-resolution: 120
nlags: 12
nugget: 0.10
sill: 1.0
range: 30
method: ordinary
variogram: spherical
```

#### 连续地形精细建模（100~400点）
```
grid-resolution: 180
nlags: 16
nugget: 0.08
sill: 1.2
range: 42
method: universal
variogram: gaussian
```

#### 大数据性能优先（> 400点）
```
grid-resolution: 90
nlags: 10
nugget: 0.20
sill: 1.2
range: 35
method: block
variogram: exponential
```

### 参数调优建议

1. **nugget参数**
   - 代表短尺度随机性或测量噪声
   - 过大会导致局部跳变
   - 建议范围：0.05-0.25

2. **sill参数**
   - 代表总体方差上限
   - 决定半方差平台高度
   - 建议范围：0.8-1.5

3. **range参数**
   - 代表空间相关影响距离
   - 过小导致图面碎片化，过大可能过度平滑
   - 建议为空间对角线长度的15%-40%

4. **grid-resolution**
   - 影响输出网格的精细程度
   - 过高会增加计算时间
   - 建议根据数据密度和精度要求调整

## 常见问题

### Q1: 插值结果出现明显的边界效应怎么办？

A: 可以尝试以下方法：
- 增加边界区域的采样点密度
- 使用边界缓冲区技术
- 调整range参数，适当增大空间相关范围

### Q2: 交叉验证误差过大如何优化？

A: 建议按以下步骤检查：
- 检查数据质量，剔除异常值
- 尝试不同的变异函数模型
- 调整nugget和sill参数
- 考虑使用更复杂的插值方法

### Q3: 大数据集处理速度慢如何提升？

A: 可以采用以下优化策略：
- 降低grid-resolution
- 使用块克里金方法
- 启用空间分块计算
- 考虑使用GPU加速

### Q4: 如何选择合适的变异函数模型？

A: 根据数据特征选择：
- 球状模型：大多数情况下的默认选择
- 指数模型：适合局地突变数据
- 高斯模型：适合连续平滑变化的数据
- 线性模型：快速探索用，不建议作为最终结果

### Q5: 插值结果的可信度如何评估？

A: 通过以下指标评估：
- 交叉验证误差（RMSE、MAE）
- 变异函数拟合质量（R²）
- 不确定性分布图
- 与已知验证点对比
