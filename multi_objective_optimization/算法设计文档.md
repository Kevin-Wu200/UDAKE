# 多目标优化采样系统 - 算法设计文档

## 1. 算法选型

### 1.1 多目标优化算法对比

| 算法 | 优点 | 缺点 | 适用场景 |
|-----|------|------|---------|
| NSGA-II | 分布均匀、收敛快、易于实现 | 复杂度高（O(MN^2)） | 中小规模问题 |
| MOEA/D | 计算效率高、可扩展性强 | 需要预设权重向量 | 大规模问题 |
| SPEA2 | 保持外部档案、精度高 | 内存占用大 | 精度要求高 |
| NSGA-III | 支持多目标（>3） | 参数设置复杂 | 高维目标问题 |

**选择结果**：NSGA-II（主要算法）+ MOEA/D（备用算法）

### 1.2 NSGA-II算法原理

NSGA-II（Non-dominated Sorting Genetic Algorithm II）是一种基于非支配排序的多目标进化算法。

**核心思想**：
1. 快速非支配排序
2. 拥挤度计算
3. 拥挤度比较算子

**算法流程**：
```
1. 初始化种群 P0
2. for t = 0 to T:
   a. 交叉变异生成子代 Qt
   b. 合并种群 Rt = Pt ∪ Qt
   c. 非支配排序 Rt
   d. 选择前N个个体作为 Pt+1
   e. 如果同一等级个体超过剩余名额，用拥挤度选择
3. 返回帕累托前沿
```

### 1.3 算法复杂度分析

| 操作 | 时间复杂度 | 空间复杂度 |
|-----|----------|----------|
| 非支配排序 | O(MN^2) | O(N) |
| 拥挤度计算 | O(MN log N) | O(N) |
| 交叉变异 | O(N) | O(N) |
| 总体 | O(MN^2) | O(N) |

其中：
- M：目标函数数量
- N：种群规模

## 3. 目标函数设计

### 3.1 方差最小化目标函数

**数学表达**：
```
f1(x) = Σ_i Var(x_i) / n
```

**实现方法**：
1. 计算每个采样点的克里金方差
2. 求和并归一化

**伪代码**：
```python
def variance_objective(individual):
    variances = []
    for point_idx in individual.genes:
        variance = kriging_variance[point_idx]
        variances.append(variance)
    return np.mean(variances)
```

### 3.2 成本最小化目标函数

**数学表达**：
```
f2(x) = w_distance * Σ_i d(x_i, base) + w_time * Σ_i t(x_i) + w_resource * Σ_i r(x_i)
```

**参数**：
- `w_distance`：距离权重
- `w_time`：时间权重
- `w_resource`：资源权重

**实现方法**：
1. 计算每个采样点到基地的距离
2. 考虑地形难度计算时间成本
3. 估算资源消耗
4. 加权求和

### 3.3 可达性最大化目标函数

**数学表达**：
```
f3(x) = -Σ_i (d(x_i, road) + difficulty(x_i))
```

**实现方法**：
1. 计算每个采样点到最近道路的距离
2. 评估地形难度
3. 求和并取负（因为是最大化）

## 4. 约束条件设计

### 4.1 边界约束

**数学表达**：
```
g1(x) = Σ_i (1 - I(x_i ∈ boundary))
```

**实现方法**：
1. 检查每个采样点是否在边界内
2. 使用点在多边形内算法

### 4.2 距离约束

**数学表达**：
```
g2(x) = Σ_i Σ_j max(0, min_distance - d(x_i, x_j))
```

**实现方法**：
1. 计算所有采样点对之间的距离
2. 统计违反最小间距的数量

### 4.3 预算约束

**数学表达**：
```
g3(x) = max(0, total_cost - budget)
```

**实现方法**：
1. 计算总成本
2. 与预算比较

## 5. NSGA-II算法实现

### 5.1 快速非支配排序

```python
def fast_non_dominated_sort(population: Population) -> List[List[int]]:
    """
    快速非支配排序
    返回：各前沿面的索引列表
    """
    fronts = [[]]
    domination_count = [0] * population.size
    dominated_solutions = [[] for _ in range(population.size)]

    for i in range(population.size):
        for j in range(population.size):
            if dominates(population.individuals[i], population.individuals[j]):
                dominated_solutions[i].append(j)
            elif dominates(population.individuals[j], population.individuals[i]):
                domination_count[i] += 1

        if domination_count[i] == 0:
            population.individuals[i].rank = 0
            fronts[0].append(i)

    k = 0
    while len(fronts[k]) > 0:
        next_front = []
        for i in fronts[k]:
            for j in dominated_solutions[i]:
                domination_count[j] -= 1
                if domination_count[j] == 0:
                    population.individuals[j].rank = k + 1
                    next_front.append(j)
        k += 1
        fronts.append(next_front)

    return fronts[:-1]  # 移除最后一个空列表
```

### 5.2 拥挤度计算

```python
def calculate_crowding_distance(front: List[int], population: Population):
    """
    计算前沿面的拥挤度
    """
    for i in front:
        population.individuals[i].crowding_distance = 0

    num_objectives = len(population.individuals[front[0]].objectives)

    for m in range(num_objectives):
        # 按目标m排序
        sorted_front = sorted(front, key=lambda i: population.individuals[i].objectives[m])

        # 边界点拥挤度设为无穷大
        population.individuals[sorted_front[0]].crowding_distance = float('inf')
        population.individuals[sorted_front[-1]].crowding_distance = float('inf')

        # 计算中间点的拥挤度
        obj_range = (population.individuals[sorted_front[-1]].objectives[m] -
                    population.individuals[sorted_front[0]].objectives[m])

        if obj_range == 0:
            continue

        for i in range(1, len(sorted_front) - 1):
            distance = (population.individualives[sorted_front[i+1]].objectives[m] -
                       population.individuals[sorted_front[i-1]].objectives[m])
            population.individuals[sorted_front[i]].crowding_distance += distance / obj_range
```

### 5.3 选择操作

```python
def selection(population: Population, n_selected: int) -> List[Individual]:
    """
    拥挤度比较算子选择
    """
    # 非支配排序
    fronts = fast_non_dominated_sort(population)

    # 计算拥挤度
    for front in fronts:
        calculate_crowding_distance(front, population)

    # 选择个体
    selected = []
    current_rank = 0

    while len(selected) + len(fronts[current_rank]) <= n_selected:
        selected.extend([population.individuals[i] for i in fronts[current_rank]])
        current_rank += 1

    # 如果最后一前沿个体超过剩余名额，按拥挤度选择
    if len(selected) < n_selected:
        remaining = n_selected - len(selected)
        sorted_front = sorted(fronts[current_rank],
                             key=lambda i: population.individuals[i].crowding_distance,
                             reverse=True)
        selected.extend([population.individuals[i] for i in sorted_front[:remaining]])

    return selected
```

### 5.4 交叉操作

```python
def crossover(parent1: Individual, parent2: Individual, pc: float = 0.9) -> Tuple[Individual, Individual]:
    """
    交叉操作：使用部分映射交叉（PMX）
    """
    if np.random.random() > pc:
        return parent1, parent2

    n = len(parent1.genes)
    child1_genes = parent1.genes.copy()
    child2_genes = parent2.genes.copy()

    # 随机选择两个交叉点
    i, j = sorted(np.random.choice(n, 2, replace=False))

    # 交换中间段
    child1_genes[i:j] = parent2.genes[i:j]
    child2_genes[i:j] = parent1.genes[i:j]

    # 修复冲突
    child1_genes = resolve_conflicts(child1_genes, parent1.genes, i, j)
    child2_genes = resolve_conflicts(child2_genes, parent2.genes, i, j)

    child1 = Individual()
    child2 = Individual()
    child1.genes = child1_genes
    child2.genes = child2_genes

    return child1, child2
```

### 5.5 变异操作

```python
def mutation(individual: Individual, pm: float = 0.1, n_candidates: int = None) -> Individual:
    """
    变异操作：随机交换
    """
    mutated = Individual()
    mutated.genes = individual.genes.copy()

    n = len(mutated.genes)

    # 随机选择变异位置
    for i in range(n):
        if np.random.random() < pm:
            # 随机替换为新的采样点
            if n_candidates is not None:
                new_gene = np.random.randint(0, n_candidates)
                mutated.genes[i] = new_gene

    return mutated
```

### 5.6 主循环

```python
def nsga2_optimizer(
    objectives: List[ObjectiveFunction],
    constraints: List[Constraint],
    population_size: int = 100,
    n_generations: int = 100,
    crossover_prob: float = 0.9,
    mutation_prob: float = 0.1
) -> Population:
    """
    NSGA-II主优化循环
    """
    # 初始化种群
    population = Population(population_size)
    initialize_population(population, n_candidates, n_samples)

    # 评估初始种群
    evaluate_population(population, objectives, constraints)

    # 进化循环
    for gen in range(n_generations):
        # 选择父代
        parents = selection(population, population_size)

        # 交叉变异生成子代
        offspring = []
        for i in range(0, population_size, 2):
            child1, child2 = crossover(parents[i], parents[i+1], crossover_prob)
            child1 = mutation(child1, mutation_prob)
            child2 = mutation(child2, mutation_prob)
            offspring.extend([child1, child2])

        # 评估子代
        for child in offspring:
            evaluate_individual(child, objectives, constraints)

        # 合并种群
        combined = population + offspring

        # 选择下一代
        next_population = select_next_generation(combined, population_size)
        population = next_population

        # 记录收敛历史
        record_convergence(gen, population)

    return population
```

```python
from multiprocessing import Pool

def parallel_evaluate_population(population: Population, objectives: List[ObjectiveFunction], n_workers: int = 4):
    """
    并行评估种群
    """
    with Pool(n_workers) as pool:
        objectives_values = pool.map(
            lambda ind: evaluate_objectives(ind, objectives),
            population.individuals
        )

    for i, obj_values in enumerate(objectives_values):
        population.individuals[i].objectives = obj_values
```

```python
def vectorized_variance_evaluation(population: Population) -> np.ndarray:
    """
    向量化评估方差目标
    """
```
