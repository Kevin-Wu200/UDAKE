## Agent skills

### Issue tracker

Issues live in GitHub Issues on `Kevin-Wu200/UDAKE`. See `docs/agents/issue-tracker.md`.

### Triage labels

Uses the canonical five triage labels: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context layout — one `CONTEXT.md` at root, one `docs/adr/` directory. See `docs/agents/domain.md`.

## AGENT RULES

### Language

 - 默认使用简体中文回答
 - 所有解释、代码说明、分析过程均使用中文
 - 仅在用户明确要求时使用英文
 - 代码注释优先使用中文
 - Git Commit message 使用中文
