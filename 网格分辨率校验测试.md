## 校验规则

### 正则表达式
```javascript
/^[1-9]\d*$/
```

### 规则说明
1. ✅ 不能为空
2. ✅ 只能为整数
3. ✅ 必须大于 0
4. ✅ 禁止小数
5. ✅ 禁止负数
6. ✅ 禁止科学计数法
7. ✅ 禁止字母
8. ✅ 禁止空格
9. ✅ 禁止前导 + 或 -
10. ✅ 限制最大值为 10000
